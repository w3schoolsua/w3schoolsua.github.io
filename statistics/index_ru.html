Обучение на примерах

В нашем редакторе "Try it Yourself" ("Попробуйте сами") вы можете использовать модули Python и код R, а также изменять код, чтобы увидеть результат.

Пример

В Python используйте метод mean() библиотеки NumPy, чтобы найти среднее значение 4, 11, 7, 14:

import numpy

values = [4,11,7,14]

x = numpy.mean(values)

print(x)
