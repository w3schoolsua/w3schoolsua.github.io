Aprendiendo con ejemplos

En nuestro editor "Try it Yourself" ("Pruébelo usted mismo"), puede usar módulos de Python y código R, y modificar el código para ver el resultado.

Ejemplo

Con Python, use el método mean() de la biblioteca NumPy para encontrar la media de los valores 4, 11, 7, 14:

import numpy

values = [4,11,7,14]

x = numpy.mean(values)

print(x)
