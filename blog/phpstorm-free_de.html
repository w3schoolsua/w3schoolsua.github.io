PHPStorm. Wie kann ich es kostenlos nutzen? Ewige Testversion

❮ Prev Next ❯

Original: "PHPStorm. Як користуватись безкоштовно? Вічний тріал",
by WebSunSey updated on 14.07.2025

Wie kann ich die PHPStorm IDE kostenlos nutzen?

PHPStorm – ist eine leistungsstarke IDE (Integrated Development Environment) von JetBrains zur Entwicklung von Webanwendungen. Sie unterstützt unter anderem PHP, HTML, CSS und JavaScript. PHPStorm bietet zahlreiche Funktionen wie automatische Codevervollständigung, Fehlerprüfung, Code-Debugging und vieles mehr.

Sie können PHPStorm von der offiziellen JetBrains-Website herunterladen.

PHPStorm-Kosten

PHPStorm ist ein kostenpflichtiges Produkt (249,00 $/Jahr). JetBrains bietet jedoch eine 30-tägige kostenlose Testversion an. Nach Ablauf der Testphase müssen Sie eine Lizenz erwerben, um PHPStorm weiterhin nutzen zu können.

Sie können PHPStorm 30 Tage lang kostenlos testen. Um die Testphase zu starten, wählen Sie die Option „Testversion starten“ im Dialogfeld „Lizenzen“, das sich beim Start von PHPStorm öffnet.

Wenn Sie ToolBox bereits zur Installation von JetBrains-Produkten verwenden und bereits angemeldet sind, meldet PHPStorm Sie automatisch bei Ihrem JetBrains-Konto an.

Wenn Sie die Zwei-Faktor-Authentifizierung für Ihr JetBrains-Konto verwenden, können Sie anstelle Ihres primären Kontopassworts ein von der App generiertes Passwort angeben.

Nach Ablauf der Testphase müssen Sie eine Lizenz erwerben und registrieren, um PHPStorm weiterhin nutzen zu können. Für die nächste Version von PHPStorm wird eine neue Testphase verfügbar sein.

Verwendung einer temporären E-Mail

Um die PHPStorm-Testlizenz zu aktivieren, müssen Sie sich per E-Mail auf der JetBrains-Website registrieren. Wenn Sie jedoch Ihre permanente E-Mail-Adresse verwenden (z. B. bei Gmail, Yahoo, Bing oder anderen großen Diensten), können Sie nur einen Testzeitraum von 30 Tagen nutzen und müssen anschließend erneut eine neue E-Mail-Adresse registrieren. Um dies zu vermeiden, können Sie temporäre E-Mail-Dienste nutzen, wie zum Beispiel:

  • https://temp-mail.org/
  • https://temp-mail.io/
  • https://tempmailo.com/
  • https://etempmail.net/
  • https://tempail.com/
  • https://dropmail.me/
  • https://www.mohmal.com/

und mehr...

Nach Ablauf der Testphase können Sie ein neues Konto mit einer neuen temporären E-Mail-Adresse erstellen und PHPStorm kostenlos weiternutzen. So können Sie PHPStorm lebenslang nutzen und sich ständig mit neuen temporären E-Mail-Adressen registrieren. Registrieren Sie sich einfach erneut auf der JetBrains-Website mit Ihrer neuen E-Mail-Adresse, aktivieren Sie die neue Lizenz und nutzen Sie PHPStorm 30 Tage lang weiter. Wiederholen Sie diesen Vorgang anschließend und nutzen Sie PHPStorm dauerhaft und ohne Funktionseinschränkungen!

Verwendung des Aktivierungscodes (Lizenzschlüssel/Seriennummer)

Eine weitere Möglichkeit, PHPStorm kostenlos zu nutzen, ist die Eingabe des Aktivierungscodes (Lizenzschlüssel/Seriennummer).

WARNUNG! Die Internetverbindung muss während der Aktivierung getrennt sein!

Starten Sie PHPStorm. Wenn Sie zur Eingabe einer Lizenz aufgefordert werden, öffnet sich das Fenster „Lizenzen verwalten“. Wählen Sie links PHPStorm aus und aktivieren Sie dann rechts im Fenster das Kontrollkästchen neben „Aktivierungscode“. Markieren und kopieren Sie den unten stehenden Code (klicken Sie auf das Feld „Aktivierungscode“, um ihn anzuzeigen) und fügen Sie ihn in das Feld „Aktivierungscode hier einfügen oder ablegen“ ein.

Klicken Sie anschließend in PHPStorm auf die Schaltfläche „Aktivieren“. Schließen Sie das Fenster „Lizenzen verwalten“.

Ihr PHPStorm bleibt bis zum Ablaufdatum dieser Lizenz aktiviert. Diese Aktivierungsschlüssel sind ab dem Veröffentlichungsdatum 30 Tage lang gültig und werden ständig aktualisiert.

Schließen Sie anschließend PHPStorm. Falls die Internetverbindung unterbrochen ist, starten Sie PHPStorm erneut. In den ersten 30 Sekunden nach dem Start versucht das Programm, die Lizenz (den Aktivierungsschlüssel) über seine Online-Dienste zu überprüfen. Bei einer Unterbrechung der Internetverbindung ist dies jedoch nicht möglich und PHPStorm funktioniert weiter. Anschließend können Sie die Internetverbindung wieder aktivieren und PHPStorm wie gewohnt nutzen.

WARNUNG! Die oben genannten Methoden zur Aktivierung von PHPStorm werden nur für diejenigen empfohlen, die sich den Erwerb einer kostenpflichtigen Lizenz für dieses Softwareprodukt nicht leisten können. Jeder, der die Möglichkeit hat, PHPStorm zu erwerben, kann dies tun und PHPStorm ohne Einschränkungen sicher nutzen.
