¿Cómo utilizar PHPStorm IDE gratis?

PHPStorm - es un potente IDE (entorno de desarrollo integrado) de JetBrains que se utiliza para desarrollar aplicaciones web, incluidas soporte para PHP, HTML, CSS, JavaScript y más. Ofrece una amplia gama de funciones como autocompletar código, verificación de errores, depuración de código y mucho más.

Puede descargar PHPStorm desde el sitio web oficial de JetBrains.

Coste de PHPStorm

PHPStorm es un producto pago ($249,00/año), pero JetBrains ofrece una prueba gratuita de 30 días. Una vez finalizado el período de prueba, deberá comprar una licencia para continuar usando PHPStorm.

Puedes evaluar PHPStorm gratis durante 30 días. Para iniciar un período de prueba, seleccione la opción Iniciar prueba en el cuadro de diálogo Licencias que se abre cuando inicia PHPStorm.

Si ya está utilizando ToolBox para instalar productos JetBrains y ya ha iniciado sesión, PHPStorm iniciará sesión automáticamente en su cuenta JetBrains.

Si utiliza la autenticación de dos factores para su cuenta JetBrains, puede especificar una contraseña generada por la aplicación en lugar de la contraseña de su cuenta principal.

Una vez finalizado el período de prueba, deberá comprar y registrar una licencia para continuar usando PHPStorm. Un nuevo período de prueba estará disponible para la próxima versión de PHPStorm.

Uso de correo electrónico temporal

Para activar la licencia de prueba de PHPStorm, debe registrarse en el sitio web de JetBrains mediante el correo electrónico. Pero si usa su correo electrónico permanente (por ejemplo, en Gmail, Yahoo, Bing u otros servicios grandes), solo podrá usar un período de prueba de 30 días y luego tendrá que registrar un nuevo correo electrónico nuevamente. Para no hacerlo, puedes utilizar servicios de correo electrónico temporales, como por ejemplo:

  • https://temp-mail.org/
  • https://temp-mail.io/
  • https://tempmailo.com/
  • https://etempmail.net/
  • https://tempail.com/
  • https://dropmail.me/
  • https://www.mohmal.com/

y otros...

Después de que finalice el período de prueba, puede crear una nueva cuenta con un nuevo correo electrónico temporal y continuar usando PHPStorm de forma gratuita. Este enfoque le permite utilizar PHPStorm de por vida, registrándose constantemente con nuevos correos electrónicos temporales. Simplemente, vuelva a registrarse en el sitio web de JetBrains con el nuevo correo electrónico que desea utilizar para el registro, active la nueva licencia y continúe usando PHPStorm durante los próximos 30 días. ¡Entonces podrás repetir este procedimiento en el futuro y usar PHPStorm para siempre sin ninguna limitación funcional!

Uso del código de activación (clave de licencia/número de serie)

Otra forma de tener la oportunidad de usar PHPStorm gratis es ingresar el Código de Activación (clave de licencia/número de serie).

¡ADVERTENCIA! ¡Internet debe estar desconectado durante la activación!

Inicie PHPStorm. Cuando se le solicite una licencia, se abrirá la ventana "Administrar licencias". A la izquierda, seleccione PHPStorm y, a la derecha de esta ventana, marque la casilla "Código de activación". Resalte y copie el código a continuación (haga clic en el panel "Código de activación" para verlo) y péguelo en el campo "Pegar o soltar el código de activación aquí".
