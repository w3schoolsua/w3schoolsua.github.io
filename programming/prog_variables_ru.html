Переменные в программировании

Переменные — это одни из самых базовых и важных понятий в программировании, используемые для хранения значений.

Что такое переменная?

У переменной есть имя, и вы можете хранить в ней какое‑то значение.

На изображении ниже показано, как можно представить себе переменную с именем favFruit и значением 'apple', которое хранится внутри неё.


Ниже показано, как мы можем создать переменную favFruit с помощью Python‑кода:

Приведённый выше код создаёт переменную с именем favFruit, а знак равенства = используется для сохранения значения 'apple' внутри переменной.

Причина присвоения переменной имени заключается в том, чтобы можно было использовать её позже в коде и понимать, какое значение она хранит.

Создание переменной

Ниже приведён код для создания переменной favAnimal на разных языках программирования.


Python: 
favAnimal = 'turtles'
JavaScript: 
const favAnimal = 'turtles';
Java: 
String favAnimal = "turtles";
C++: 
string fav_animal = "turtles";

Переменные могут хранить разные типы данных, такие как целые числа, десятичные числа или текст.

Примечание: При создании переменной в таких языках программирования, как C/C++ и Java, мы должны сообщить компьютеру, какой тип данных содержит переменная. Для этого нужно написать, например, int перед именем переменной, если переменная содержит целое число.

Работа с переменными

Как мы видели в предыдущем примере, значение может храниться в переменной. А если вы запустите приведённый выше пример кода, то увидите, как переменная выводится.

Мы также можем выполнять другие действия с переменными, такие как математические операции или объединение переменных с текстовыми строками.

Добавление переменной к строке

Чтобы использовать переменную в строке, вы можете добавить её к строке вот так:


Python: 
a = 'Jane'
print('Hello, my name is ' + a)
JavaScript: 
const a = 'Jane';
console.log('Hello, my name is ' + a);
Java: 
String a = "Jane";
System.out.println("Hello, my name is " + a);
C++: 
string a = "Jane";
cout << "Hello, my name is " + a;

Добавление двух строковых переменных вместе

Вы можете сложить две строковые переменные, чтобы получить предложение, используя оператор +, вот так:


Python: 
a = 'Jane'
b = 'My name is '
print(b + a)
JavaScript: 
const a = 'Jane';
const b = 'My name is ';
console.log(b + a);
Java: 
String a = "Jane";
String b = "My name is ";
System.out.println(b + a);
C++: 
string a = "Jane";
string b = "My name is ";
cout << b + a;

int c = a + b;
cout << "The sum a + b is " + to_string(c) + "\\n";

console.log(a);
Java: 
int a = 2;
a = a + 1;
System.out.println(a);
C++:

a = a + 1;
cout << a;

Использование переменной в инструкции if

Мы можем использовать переменную в инструкции if как часть условия, вот так:


Python: 
temperature = 25
print('Temperature: ' + str(temperature) + '°C')

if temperature > 20:
  print('It is warm')
else:
  print('It is not warm')
JavaScript: 
const temperature = 25;
console.log('Temperature: ' + temperature + '°C');

if (temperature > 20) {
  console.log('It is warm');
} else {
  console.log('It is not warm');
}
Java: 
int temperature = 25;
System.out.println("Temperature: " + temperature + "°C");

if (temperature > 20) {
  System.out.println("It is warm");
} else {
  System.out.println("It is not warm");
}
C++: 
int temperature = 25;
cout << "Temperature: " + to_string(temperature) + "°C\\n";

if (temperature > 20) {
  cout << "It is warm\\n";
} else {
  cout << "It is not warm\\n";
}

Имя переменной

Существуют определённые правила, которые применяются при именовании переменных. Некоторые правила специфичны для отдельных языков программирования, другие — общие для всех языков:

  • Имя переменной не может содержать пробелы.
  • Имя переменной не может начинаться с числа.
  • Имя переменной не может быть зарезервированным словом, таким как if, else, for, function и т.д.

Для удобства чтения обычно используется camelCase или snake_case при именовании переменных, поэтому вместо myfavanimal мы можем использовать myFavAnimal или my_fav_animal.
