HTML

Язык для создания веб-страниц

HTML - язык гипертекстовой разметки, предназначенный для создания веб-страниц. HTML не является языком программирования. С помощью html-тегов производится разметка на веб-странице, с помощью которой размещаются разные объекты веб-документа: заголовки, текст, изображения, таблицы, списки, фреймы, видеоплееры и т.д.

CSS

Язык для стилизации веб-страниц

CSS (каскадные таблицы стилей) – язык, отвечающий за визуальное представление информации и оформление веб-страницы. С помощью стилей производится форматирование текста и другая стилизация (оформление) объектов на веб-странице. CSS обычно используется в паре с HTML.

JavaScript

Язык для программирования веб-страниц

JavaScript (JS) - язык программирования, третья составляющая (вместе с HTML и CSS) стандартных веб-технологий, применяемых для создания веб-сайтов. JavaScript превращает статические веб-страницы в динамические, изменяющиеся и взаимодействующие с пользователем. Язык программирования JavaScript управляет мультимедиа, анимирует изображения, изменяет текст на веб-странице и многое другое.

Python

Популярна мова програмування

Python (Пайтон/Пітон) - високорівнева мова програмування загального призначення, орієнтована на підвищення продуктивності розробника і читання коду. Застосовується для програмування як загальних застосунків, так і в вебпрограмуванні (створення сайтів). Рекомендується для вивчення початківцями, що вивчають вебпрограмування, а також як перша мова програмування для новачків завдяки своїй простоті та доступності.

PHP

Мова веб-сервера

PHP (препроцесор гіпертексту) - скриптова мова загального призначення, застосовується для розробки вебдодатків. Підтримується абсолютною більшістю хостинг-провайдерів і є однією з основних мов, які застосовуються для створення динамічних вебсайтів. За допомогою PHP часто створюють CMS - систему управління контентом (рушій сайту). PHP працює на стороні сервера (backend), забезпечуючи взаємодію з користувачем. На PHP створені такі сайти, як Facebook, Yahoo, Вікіпедія, рушій Wordpress та ін. Мову програмування PHP ви можете вивчити онлайн на сайті W3Schools українською.
