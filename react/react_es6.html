Що таке ES6?

ES6 означає ECMAScript 6.

ECMAScript було створено для стандартизації JavaScript, а ES6 — це 6-та версія ECMAScript, опублікована в 2015 році, також відома як ECMAScript 2015.

Чому я маю вивчати ES6?

React використовує ES6, і ви повинні бути знайомі з деякими новими функціями, як-от:

  • Класи
  • Функції стрілок
  • Змінні (let, const, var)
  • Методи масиву, наприклад .map()
  • Деструктуризація
  • Модулі
  • Тернарний оператор
  • Оператор розповсюдження
