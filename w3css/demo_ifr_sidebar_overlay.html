Наложение сайдбара

Класс w3-overlay можно использовать для создания эффекта наложения при открытии боковой панели. Класс w3-overlay добавляет черный фон с 50% непрозрачности к "странице контента" - этот эффект будет "подсвечивать" боковую навигацию.

Нажмите на меню "гамбургер" чтобы увидеть эффект.
