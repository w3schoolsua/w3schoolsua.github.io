Сторінки зі стилем W3.CSS легко читати!

Сторінки, стилізовані за допомогою каскадної таблиці стилів W3.CSS легко читаються, навіть для людей з невеликими порушеннями зору.

  • В W3.CSS за замовчуванням розмір шрифту складає 15 пікселів
  • За замовчуванням використовується шрифт Verdana зі зручним міжлітерним інтервалом
  • Міжрядковий інтервал за замовчуванням (1.5) також дуже зручний

HTML Заголовки <h1> - <h6>

За замовчуванням W3.CSS стилізує HTML заголовки таким чином:

Приклад

<h1>Заголовок 1 (36px)</h1>

<h2>Заголовок 2 (30px)</h2>

<h3>Заголовок 3 (24px)</h3>

<h4>Заголовок 4 (20px)</h4>

<h5>Заголовок 5 (18px)</h5>
<h6>Заголовок 6 (16px)</h6>

Класи розміру шрифта (font-size)

Заголовки мають використовуватись як є: тільки як заголовки.

В W3.CSS розміри шрифту можуть бути встановлені за допомогою наступних класів w3-size:

Приклад

w3-tiny
w3-small
w3-medium (за замовчуванням)
w3-large
w3-xlarge
w3-xxlarge
w3-xxxlarge
w3-jumbo

Перевизначення налаштувань W3.CSS за замовчуванням

Перевизначити налаштування W3.CSS за замовчуванням дуже просто.

Є як мінімум три варіанти:

  1. Перевизначить налаштування за замовчуванням в розділі <head> веб-сторінки
  2. Додайте посилання на свій власний файл CSS після посилання на файл W3.CSS
  3. Завантажте та змініть зміст файлу W3.CSS на свій розсуд

Перевизначити в розділі <head>

h1{font-size:64px;}
h2{font-size:48px;}
h3{font-size:36px;}
h4{font-size:24px;}
h5{font-size:20px;}
h6{font-size:18px;}
Спробуйте самі »

Додати посилання на свій власний файл CSS

<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
<link rel="stylesheet" href="myStyle.css"> <!-- посилання на свій файл css -->

Змінити файл W3.CSS

h1{font-size:36px} h2{font-size:30px} h3{font-size:24px} h4{font-size:20px} h5{font-size:18px} h6{font-size:16px}

Зміна шрифту сторінки за замовчуванням

В заголовку вашої веб-сторінки (або в таблиці стилів) після підключення W3.CSS змініть стиль html та body:

Приклад

<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
<style>
html, body, h1, h2, h3, h4, h5, h6 {
  font-family: "Comic Sans MS", cursive, sans-serif;
}
</style>

W3.CSS Зовнішні шрифти

Making the web beautiful!

За допомогою W3.CSS дуже легко додавати нові шрифти на вебсторінку.

  • Дуже просто у використанні (лише CSS та HTML)
  • Необмежене використання зовнішніх бібліотек шрифтів (наприклад, Google Fonts)
  • Працює в усіх сучасних браузерах

Використання класу шрифтів

Створюємо мережу разом!

В заголовку вебсторінки (або в таблиці стилів) створіть клас шрифту:

Приклад

.w3-myfont {
  font-family: "Comic Sans MS", cursive, sans-serif;
}

В телі (body) вашої вебсторінки використовуйте ім’я класу шрифта:

Приклад

<body>
  <p class="w3-myfont">Створюємо мережу разом!</p>
</body>
Спробуйте самі »

Використання зовнішніх шрифтів

В заголовку вебсторінки додайте зовнішній шрифт та присвойте шрифту ім’я класу:

Приклад

<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Tangerine">

<style>
.w3-tangerine {
  font-family: 'Tangerine', serif;
}
</style>

В тілі (body) вебсторінки використовуйте ім’я класу:

Приклад

<body>
  <p class="w3-tangerine">Робимо Інтернет красивим!</p>
</body>
