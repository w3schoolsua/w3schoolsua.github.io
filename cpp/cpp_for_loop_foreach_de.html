Die foreach-Schleife

Es gibt auch eine „for-each“-Schleife (eingeführt in C++ Version 11 (2011), die ausschließlich zum Durchlaufen von Elementen in einem Array verwendet wird (oder andere Datensätze):

Syntax

for (type variableName : arrayName) {
  // Codeblock, der ausgeführt werden soll
}

Im folgenden Beispiel werden alle Elemente in einem Array mithilfe einer „for-each“-Schleife ausgegeben:

Beispiel

int myNumbers[5] = {10, 20, 30, 40, 50};
for (int i : myNumbers) {
  cout << i << "\n";
}
Try it Yourself »

Hinweis: Machen Sie sich keine Sorgen, wenn Sie das obige Beispiel nicht verstehen. Mehr über Arrays erfahren Sie im Kapitel C++-Arrays.
