HTML

Мова для створення веб-сторінок

HTML - мова гіпертекстової розмітки, призначена для створення вебсторінок. HTML не є мовою програмування. За допомогою html-тегів проводиться розмітка на вебсторінці, за допомогою якої розміщуються різні об'єкти вебдокумента: заголовки, текст, зображення, таблиці, списки, фрейми, відеоплеєри і т.д.

</body>
</html>

CSS

Мова для стилізації веб-сторінок

CSS (каскадні таблиці стилів) - мова, яка відповідає за візуальне представлення інформації та оформлення вебсторінки. За допомогою стилів проводиться форматування тексту та інша стилізація (оформлення) об’єктів на вебсторінці. CSS зазвичай використовується в парі з HTML.

JavaScript

Мова для програмування веб-сторінок

JavaScript (JS) - мова програмування, третя складова (разом з HTML та CSS) стандартних веб-технологій, що застосовується для створення вебсайтів. JavaScript перетворює статичні вебсторінки в динамічні, мінливі та взаємодіючі з користувачем. Мова програмування JavaScript управляє мультимедіа, анімує зображення, змінює текст на вебсторінці та багато іншого.

Python

Мова програмування

Python (Пайтон/Пітон) - високорівнева мова програмування загального призначення, орієнтована на підвищення продуктивності розробника і читання коду. Застосовується для програмування як загальних застосунків, так і в веб-програмуванні (створення сайтів). Рекомендується для вивчення початківцями, що вивчають вебпрограмування, а також як перша мова програмування для новачків завдяки своїй простоті та доступності. Мову програмування Python ви можете вивчити онлайн на сайті w3schools українською.

ВИВЧИТИ PYTHON

PHP

Мова програмування веб-сервера

PHP (препроцесор гіпертексту) - скриптова мова загального призначення, застосовується для розробки вебдодатків. Підтримується абсолютною більшістю хостинг-провайдерів і є однією з основних мов, які застосовуються для створення динамічних вебсайтів. За допомогою PHP часто створюють CMS - систему управління контентом (рушій сайту). PHP працює на стороні сервера (backend), забезпечуючи взаємодію з користувачем. На PHP створені такі сайти, як Facebook, Yahoo, Вконтакті, Вікіпедія, рушій Wordpress та ін. Мову програмування PHP ви можете вивчити онлайн на сайті w3schools українською.

ВИВЧИТИ PHP

jQuery

Бібліотека JavaScript для розробки веб-сторінок

jQuery - популярна JavaScript бібліотека з відкритим кодом. Синтаксис jQuery розроблений, щоб зробити орієнтування в навігації зручним завдяки вибору елементів DOM, створення анімації, обробки подій і розробці AJAX-додатків. Допомагає створенню потужних динамічних вебсторінок. Пройти повний курс онлайн по бібліотеці jQuery ви можете на сайті w3schools українською.
