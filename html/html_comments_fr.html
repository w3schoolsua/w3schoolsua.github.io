HTML Commentaires

❮ Prev Next ❯

Les commentaires HTML ne sont pas affichés dans le navigateur, mais ils peuvent vous aider à documenter votre code source HTML.

HTML Balise de commentaire

Vous pouvez ajouter des commentaires à votre source HTML en utilisant la syntaxe suivante :

<!-- Écrivez vos commentaires ici -->

Notez qu'il y a un point d'exclamation (!) dans la balise de début, mais pas dans la balise de fin.

Remarque : Les commentaires ne sont pas affichés par le navigateur, mais ils peuvent aider à documenter votre code source HTML.

Ajouter des commentaires

Avec les commentaires, vous pouvez placer des notifications et des rappels dans votre code HTML :

Exemple

<!-- Ceci est un commentaire -->

<p>Ceci est un paragraphe.</p>

<!-- N'oubliez pas d'ajouter plus d'informations ici -->
Try it Yourself »

Masquer le contenu

Les commentaires peuvent être utilisés pour masquer le contenu.

Ce qui peut être utile si vous masquez temporairement du contenu :

Exemple

<p>Ceci est un paragraphe.</p>

<!-- <p>Ceci est un autre paragraphe </p> -->

<p>C'est aussi un paragraphe.</p>
Try it Yourself »

Vous pouvez également masquer plus d'une ligne, tout ce qui se trouve entre le <!-- et le --> sera être masqué de l’écran.

Exemple

Masquer une section de code HTML :

<p>Ceci est un paragraphe.</p>
<!--
<p>Regardez cette image sympa :</p>
<img border="0" src="pic_trulli.jpg" alt="Trulli">
-->
<p>C'est aussi un paragraphe.</p>
Try it Yourself »

Les commentaires sont également parfaits pour déboguer le code HTML, car vous pouvez commenter les lignes de code HTML, une à la fois, pour rechercher des erreurs.

Masquer le contenu en ligne

Les commentaires peuvent être utilisés pour masquer des parties au milieu du code HTML.

Exemple

Masquer une partie d'un paragraphe :

<p>Ce <!-- super texte --> est un paragraphe.</p>
Try it Yourself »
