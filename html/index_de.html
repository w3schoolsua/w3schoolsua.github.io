Einfaches Lernen mit HTML „Probieren Sie es selbst aus“

Mit unserem „Try it Yourself“ („Versuch es selber“) - Editor können Sie den HTML-Code bearbeiten und das Ergebnis ansehen:

Beispiel

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
Versuch es selber »

Klicken Sie auf die Schaltfläche „Try it Yourself“, um zu sehen, wie es funktioniert.
