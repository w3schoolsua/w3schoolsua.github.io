HTML Iframes

❮ Prev Next ❯

Un iframe HTML est utilisé pour afficher une page Web dans une page Web.

Syntaxe HTML Iframe

La balise HTML <iframe> spécifie un cadre en ligne.

Un cadre en ligne est utilisé pour intégrer un autre document dans le document HTML actuel.

Syntaxe

<iframe src="url" title="description"></iframe>

Astuce : Il est recommandé de toujours inclure un attribut title pour le <iframe>. Ceci est utilisé par les lecteurs d'écran pour lire le contenu de l'iframe.

Iframe - Définir la hauteur et la largeur

Utilisez les attributs height et width pour spécifier la taille de l'iframe.

La hauteur et la largeur sont spécifiées en pixels par défaut :

Exemple

<iframe src="demo_iframe.htm" height="200" width="300" title="Iframe Exemple"></iframe>
Try it Yourself »

Ou vous pouvez ajouter l'attribut style et utiliser les CSS height et width Propriétés :

Exemple

<iframe src="demo_iframe.htm" style="height:200px;width:300px;" title="Iframe Exemple"></iframe>
Try it Yourself »

Iframe - Supprimer la bordure

Par défaut, une iframe est entourée d’une bordure.

Pour supprimer la bordure, ajoutez l'attribut style et utilisez la propriété CSS border :

Exemple

<iframe src="demo_iframe.htm" style="border:none;" title="Iframe Exemple"></iframe>
Try it Yourself »

Avec CSS, vous pouvez également modifier la taille, le style et la couleur de la bordure de l'iframe :

Exemple

<iframe src="demo_iframe.htm" style="border:2px solid red;" title="Iframe Exemple"></iframe>
Try it Yourself »

Iframe — Cible pour un lien

Une iframe peut être utilisée comme cadre cible pour un lien.

L'attribut target du lien doit faire référence à l'attribut name de l'iframe :

Exemple

<iframe src="demo_iframe.htm" name="iframe_a" title="Iframe Exemple"></iframe>

<p><a href="https://www.w3schools.com" target="iframe_a">W3Schools.com</a></p>
Try it Yourself »

Résumé du chapitre

  • La balise HTML <iframe> spécifie un cadre en ligne
  • L'attribut src définit l'URL de la page à intégrer
  • Toujours inclure un attribut title (pour les lecteurs d'écran)
  • Les attributs height et width spécifient la taille de l'iframe
  • Utilisez border:none; pour supprimer la bordure autour de l'iframe

📚 Ressources et spécifications officielles

Pour tout savoir sur l'élément `<iframe>`, nous vous recommandons de consulter les sources originales et la documentation officielle :

  • MDN Web Docs: HTML <iframe> element — La meilleure ressource pour les développeurs, avec des exemples, des descriptions d'attributs et des informations sur la compatibilité navigateurs.
  • WHATWG HTML Living Standard: The <iframe> element — La spécification technique la plus récente qui définit le fonctionnement détaillé de l'élément.
  • W3C HTML5 Specification: The <iframe> element — La norme classique (bien que le secteur privilégie désormais la norme évolutive du WHATWG).
