HTML Tableau Colspan & Rowspan

❮ Prev Next ❯

Les tableaux HTML peuvent contenir des cellules qui s'étendent sur plusieurs lignes et/ou colonnes.

NAME  
     
     
     
     
APRIL    
   
   
     
     
2025
     
FIESTA  
 
     

Tableau HTML - Colspan

Pour qu'une cellule s'étende sur plusieurs colonnes, utilisez l'attribut colspan :

Exemple

<table>
  <tr>
    <th colspan="2">Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>43</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>57</td>
  </tr>
</table>
Try it Yourself »

Remarque : La valeur de l'attribut colspan représente le nombre de colonnes à couvrir.

Tableau HTML - Portée des lignes

Pour qu'une cellule s'étende sur plusieurs lignes, utilisez l'attribut rowspan :

Exemple

<table>
  <tr>
    <th>Name</th>
    <td>Jill</td>
  </tr>
  <tr>
    <th rowspan="2">Phone</th>
    <td>555-1234</td>
  </tr>
  <tr>
    <td>555-8745</td>
</tr>
</table>
Try it Yourself »

Remarque : La valeur de l'attribut rowspan représente le nombre de lignes à s'étendre.
