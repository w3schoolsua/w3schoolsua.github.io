Was ist Kotlin?

Kotlin ist eine moderne, trendige Programmiersprache.

Kotlin ist leicht zu erlernen, insbesondere wenn Sie bereits Java kennen (es ist 100 % kompatibel mit Java).

Kotlin wird zur Entwicklung von Android-Apps, serverseitigen Apps und vielem mehr verwendet.

Beginnen Sie jetzt mit dem Erlernen von Kotlin »

Beispiele in jedem Kapitel

Unser „Try it Yourself“-Editor erleichtert das Erlernen von Kotlin. Sie können den Code bearbeiten und das Ergebnis in Ihrem Browser anzeigen:

Beispiel

fun main() {
    println("Hello World")
    }
Try it Yourself »

Klicken Sie auf die Schaltfläche „Try it Yourself“, um zu sehen, wie es funktioniert.

Wir empfehlen, dieses Tutorial in der im linken Menü aufgeführten Reihenfolge zu lesen.
