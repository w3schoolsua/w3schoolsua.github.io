Что такое Kotlin?

Kotlin — это современный, актуальный язык программирования.

Kotlin легко освоить, особенно если вы уже знаете Java (он на 100% совместим с Java).

Kotlin используется для разработки приложений для Android, серверных приложений и многого другого.

Начать изучение Kotlin немедленно! »
