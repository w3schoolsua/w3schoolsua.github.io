Назви кольорів HTML / CSS успадковані від Стандарта X11.

HTML / CSS визначає сірий колір в середині 8-бітної шкали сірого (128,128,128).

X11 визначив Gray (190,190,190); що ближче до HTML Silver.

Gray / Grey Веб-кольори

Є кілька назв сірого кольору в HTML / CSS.

Усі сірі кольори пишуться як gray (не grey).

Цей правопис був успадкований від Стандарту X11.

Усі сучасні браузери приймають і gray, і grey, але ранні версії Internet Explorer не розпізнавали grey.

16 мільйонів різних кольорів

Поєднання червоного, зеленого та синього значень від 0 до 255 дає загалом понад 16 мільйонів різних кольорів (256 x 256 x 256).

Більшість сучасних моніторів здатні відображати щонайменше 16384 різних кольорів.

Що таке веб-безпечні кольори?

Багато років тому, коли комп’ютери підтримували максимум 256 різних кольорів, з'явився список із 216 'кольорів, безпечних для веб було запропоновано як вебстандарт (з резервуванням 40 фіксованих системних кольорів).

Зараз це не важливо, оскільки більшість комп’ютерів можуть відображати мільйони різних кольорів.

Ця кольорова палітра кольорів із 216 шістнадцяткових значень у веббраузері була створена для того, щоб усі комп’ютери правильно відображали кольори під час запуску 256 кольорової палітри:
