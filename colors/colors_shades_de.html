Eine Anomalie in der obigen Tabelle besteht darin, dass HTML-Grau dunkler ist als Dunkelgrau.

Die Farbnamen von HTML / CSS wurden vom Standard X11 geerbt.

HTML / CSS definiert Grau in der Mitte der 8-Bit-Grauskala (128,128,128).

X11 definierte Grau als (190,190,190), was näher an HTML-Silber liegt.

Grau / Graue Webfarben

Es gibt mehrere graue Farbnamen in HTML / CSS.

Alle grauen Farben werden als grau (nicht grau) geschrieben.

Diese Schreibweise wurde vom Standard X11 übernommen.

Alle modernen Browser akzeptieren sowohl Grau als auch Grau, aber frühe Versionen von Internet Explorer erkannten Grau nicht.

16 Millionen verschiedene Farben

Die Kombination aus Rot-, Grün- und Blauwerten von 0 bis 255 ergibt insgesamt mehr als 16 Millionen verschiedene Farben zum Spielen (256 x 256 x 256).

Die meisten modernen Monitore sind in der Lage, mindestens 16384 verschiedene Farben darzustellen.

Websichere Farben?

Vor vielen Jahren, als Computer maximal 256 verschiedene Farben unterstützten, wurde eine Liste von 216 "Web Safe Colors" als Webstandard vorgeschlagen (Reservierung von 40 festen Systemfarben).

Das ist jetzt nicht mehr wichtig, da die meisten Computer Millionen verschiedener Farben anzeigen können.

Diese browserübergreifende Farbpalette mit 216 Hexadezimalwerten wurde erstellt, um sicherzustellen, dass alle Computer die Farben korrekt anzeigen, wenn eine 256-Farbpalette ausgeführt wird:
