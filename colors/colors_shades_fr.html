Une anomalie dans le tableau ci-dessus est que HTML Gray est plus foncé que DarkGray.

Les noms de couleurs HTML / CSS ont été hérités de la norme X11.

HTML / CSS définit le gris au milieu de l’échelle de gris de 8 bits (128,128,128).

X11 a défini le gris comme étant (190,190,190), ce qui est plus proche de l’argent HTML.

Gris / Couleurs Web grises

Il existe plusieurs noms de couleurs grises en HTML / CSS.

Toutes les couleurs grises sont orthographiées comme grises (et non grises).

Cette orthographe a été héritée de la norme X11.

Tous les navigateurs modernes acceptent à la fois le gris et le gris, mais les premières versions d’Internet Explorer ne reconnaissaient pas le gris.

16 millions de couleurs différentes

La combinaison des valeurs Rouge, Vert et Bleu de 0 à 255 donne un total de plus de 16 millions de couleurs différentes avec lesquelles jouer (256 x 256 x 256).

La plupart des moniteurs modernes sont capables d’afficher au moins 16384 couleurs différentes.

Des couleurs sécurisées pour le Web ?

Il y a de nombreuses années, lorsque les ordinateurs supportaient un maximum de 256 couleurs différentes, une liste de 216 « couleurs Web Safe » a été suggérée comme norme Web (réservant 40 couleurs système fixes).

Ce n’est plus important maintenant, car la plupart des ordinateurs peuvent afficher des millions de couleurs différentes.

Cette palette de couleurs multi-navigateur de 216 valeurs hexadécimales a été créée pour garantir que tous les ordinateurs affichent correctement les couleurs lors de l’exécution d’une palette de couleurs 256 :
