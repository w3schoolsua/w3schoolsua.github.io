Los nombres de color de HTML / CSS se heredaron del estándar X11.

HTML / CSS definió gris en el punto medio de la escala de grises de 8 bits (128,128,128).

X11 definió el gris como (190,190,190), que está más cerca del HTML plateado.

Gris / Colores Web Grises

Hay varios nombres de color gris en HTML / CSS.

Todos los colores grises se escriben como gris (no gris).

Esta ortografía fue heredada del estándar X11.

Todos los navegadores modernos aceptan tanto el gris como el gris, pero las primeras versiones de Internet Explorer no reconocían el gris.

16 millones de colores diferentes

La combinación de los valores de rojo, verde y azul de 0 a 255 da un total de más de 16 millones de colores diferentes con los que jugar (256 x 256 x 256).

La mayoría de los monitores modernos son capaces de mostrar al menos 16384 colores diferentes.

¿Colores seguros para la web?

Hace muchos años, cuando las computadoras admitían un máximo de 256 colores diferentes, se sugirió una lista de 216 "Colores seguros para la web" como estándar web (reservando 40 colores fijos del sistema).

Esto no es importante ahora, ya que la mayoría de las computadoras pueden mostrar millones de colores diferentes.

Esta paleta de colores multinavegador de 216 valores hexadecimales se creó para garantizar que todos los ordenadores mostraran los colores correctamente al ejecutar una paleta de 256 colores:
