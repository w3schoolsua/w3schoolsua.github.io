HTML

Idioma para crear páginas web

HTML - Lenguaje de marcado de hipertexto para la creación de páginas web. HTML no es un lenguaje de programación. Con la ayuda de etiquetas html, se realiza un marcado en una página web, con la ayuda del cual se colocan varios objetos de un documento web: encabezados, texto, imágenes, tablas, listas, marcos, reproductores de video, etc.

Aprender HTML HTML Referencia

HTML Ejemplo:

<!DOCTYPE html>
<html>
<head>
  <title>HTML Tutorial</title>
<head>
<body>
  <h1>Este es un título</h1>
  <p>Este es un párrafo.</p>
</body>
</html>

CSS (cascading style sheets) — Lenguaje que se encarga de la presentación visual de la información y del diseño de una página web. Los estilos se utilizan para dar formato al texto y otra estilización (diseño) de objetos en una página web. CSS normalmente se combina con HTML.

JavaScript

Lenguaje para programar páginas web

JavaScript (JS) - El lenguaje de programación, el tercer componente (junto con HTML y CSS) de las tecnologías web estándar, se utiliza para crear sitios web. JavaScript hace que las páginas web estáticas sean dinámicas, fluidas y fáciles de usar. El lenguaje de programación JavaScript administra medios, anima imágenes, cambia texto en una página web y más.

PHP

Idioma del servidor web

PHP (Preprocesador de hipertexto) es un lenguaje de programación de propósito general que se utiliza para desarrollar aplicaciones web. Cuenta con el respaldo de la gran mayoría de proveedores de hosting y es uno de los principales lenguajes utilizados para crear sitios web dinámicos. PHP se utiliza a menudo para crear un CMS, un sistema de gestión de contenidos (motor del sitio). PHP se ejecuta en el lado del servidor (backend) proporcionando interacción con el usuario. Se han creado en PHP sitios como Facebook, Yahoo, Wikipedia, el motor WordPress, etc. El lenguaje de programación PHP lo puedes Aprender online en el sitio web W3Schools en español.
