HTML

Idioma para crear páginas web

HTML - Lenguaje de marcado de hipertexto para la creación de páginas web. HTML no es un lenguaje de programación. Con la ayuda de etiquetas html, se realiza un marcado en una página web, con la ayuda de la cual se colocan varios objetos de un documento web: encabezados, texto, imágenes, tablas, listas, marcos, reproductores de video, etc.

</body>
</html>

JavaScript

Lenguaje para programar páginas web

JavaScript (JS) - el lenguaje de programación, el tercer componente (junto con HTML y CSS) de las tecnologías web estándar, se utiliza para crear sitios web. JavaScript hace que las páginas web estáticas sean dinámicas, fluidas y fáciles de usar. El lenguaje de programación JavaScript administra medios, anima imágenes, cambia texto en una página web y más.

PHP

Lenguaje de programación de servidor web

PHP (Hypertext Preprocessor) es un lenguaje de programación de propósito general utilizado para desarrollar aplicaciones web. Cuenta con el respaldo de la gran mayoría de proveedores de hosting y es uno de los principales lenguajes utilizados para crear sitios web dinámicos. PHP se utiliza a menudo para crear un CMS, un sistema de gestión de contenidos (motor del sitio). PHP se ejecuta en el lado del servidor (backend) proporcionando interacción con el usuario. En PHP se han creado sitios como Facebook, Yahoo, Vkontakte, Wikipedia, el motor WordPress, etc. El lenguaje de programación PHP que puedes Aprender online en el sitio W3Schools en español.

jQuery - popular biblioteca de código abierto JavaScript. La sintaxis jQuery está diseñada para facilitar la navegación seleccionando elementos DOM, creando animaciones, manejando eventos y desarrollando aplicaciones AJAX. Ayuda a crear potentes páginas web dinámicas. Para ver el curso en línea completo sobre la biblioteca jQuery, visite W3Schools en español.
