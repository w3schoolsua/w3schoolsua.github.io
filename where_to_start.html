Люди, які створюють вебсайти та вебпрограми, і заробляють цим собі на досить безбідне життя, називаються front-end розробниками.

Підказка: Багато front-end розробників також мають базові знання про різні фреймворки та бібліотеки CSS і JavaScript, наприклад Bootstrap, SASS (CSS pre-processor), jQuery та React,
і популярну систему контролю версій, Git.

Що про Back-End?

Front-end розробка відноситься до клієнтської сторони (як виглядає веб-сторінка).
Back-end розробка відноситься до серверної сторони (як працює веб-сторінка).

Front-end код (або Інтерфейсний код) використовується для створення статичних вебсайтів, метою яких є відображення вебсторінки. Однак, якщо ви хочете зробити свій вебсайт динамічним (керувати файлами та базами даних, додавати контактні форми, контролювати доступ користувачів тощо), вам потрібно вивчити мову програмування, як-от PHP або Python, і використовувати SQL для комунікації з базами даних.

Якщо у вас уже є певний досвід роботи з серверною частиною, ви повинні знати, що ви також можете використовувати сервіс W3Schools Spaces для створення, тестування та розгортання базового коду.
