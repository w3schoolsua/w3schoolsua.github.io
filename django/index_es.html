Django es un framework web back-end del lado del servidor.

Django es gratuito, de código abierto y está escrito en Python.

Django facilita la creación de páginas web con Python.

Empieza a aprender Django ahora ❯

Aprendiendo Haciendo

En este tutorial, obtendrás una guía paso a paso sobre cómo instalar y crear un proyecto de Django. Aprenderás a crear un proyecto donde podrás agregar, leer, actualizar o eliminar datos.

Aprenderás a crear plantillas HTML y a usar etiquetas de plantilla de Django para insertar datos en un documento HTML.

Aprenderás a trabajar con conjuntos de consultas (QuerySets) para extraer, filtrar y ordenar datos de la base de datos.

También aprenderás a configurar una base de datos PostgreSQL y a implementar tu proyecto de Django.
