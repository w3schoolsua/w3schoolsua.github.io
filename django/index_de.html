Django Tutorial

Django ist ein serverseitiges Back-End-Webframework.

Django ist kostenlos, Open Source und in Python geschrieben.

Django erleichtert die Erstellung von Webseiten mit Python.

Beginnen Sie jetzt, Django zu lernen ❯

Learning by Doing

In diesem Tutorial erhalten Sie eine Schritt-für-Schritt-Anleitung zur Installation und Erstellung eines Django-Projekts. Sie lernen, wie Sie ein Projekt erstellen, in dem Sie Daten hinzufügen, lesen, aktualisieren und löschen können.

Sie lernen, wie Sie HTML-Vorlagen erstellen und Django-Vorlagen-Tags verwenden, um Daten in ein HTML-Dokument einzufügen.

Sie lernen, wie Sie mit QuerySets Daten aus der Datenbank extrahieren, filtern und sortieren.

Sie lernen außerdem, wie Sie eine PostgreSQL-Datenbank einrichten und Ihr Django-Projekt bereitstellen.
