Django est un framework web back-end côté serveur.

Django est gratuit, open source et écrit en Python.

Django simplifie la création de pages web avec Python.

Commencez à apprendre Django maintenant ❯

Apprentissage par la pratique

Ce tutoriel vous guide pas à pas pour installer et créer un projet Django. Vous apprendrez à créer un projet dans lequel vous pourrez ajouter, lire, mettre à jour ou supprimer des données.

Vous apprendrez à créer des modèles HTML et à utiliser les balises de modèles Django pour insérer des données dans un document HTML.

Vous apprendrez à utiliser les QuerySets pour extraire, filtrer et trier les données de la base de données.

Vous apprendrez également à configurer une base de données PostgreSQL et à déployer votre projet Django.
