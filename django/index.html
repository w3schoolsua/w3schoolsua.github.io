Django — це серверна веб-платформа.

Django є безкоштовним, з відкритим кодом і написаним на Python.

Django полегшує створення вебсторінок за допомогою Python.

Почати вивчати Django негайно ❯

Навчання на практиці

У цьому посібнику ви отримаєте покрокові інструкції щодо встановлення та створення проєкту Django. Ви дізнаєтеся, як створити проєкт, у якому можна додавати, читати, оновлювати чи видаляти дані.

Ви дізнаєтеся, як створювати шаблони HTML і використовувати теги шаблонів Django для вставлення даних у документ HTML.

Ви дізнаєтеся, як працювати з QuerySets, щоб видобувати, фільтрувати та сортувати дані з бази даних.

Ви також дізнаєтеся, як налаштувати базу даних PostgreSQL і як розгорнути свій проєкт Django для всього світу.
