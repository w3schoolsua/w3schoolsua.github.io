Windows-1252

Windows-1252 був першим стандартним набором символів у Microsoft Windows.

Це був найпопулярніший набір символів у Windows з 1985 по 1990 рік.

ANSI

Історично термін "кодові сторінки ANSI" використовувався в Windows для позначення наборів символів, відмінних від DOS.

Намір полягав у тому, щоб ці набори символів відповідали стандартам ANSI, таким як ISO-8859-1.

Хоча Windows-1252 майже ідентична ISO-8859-1, вона ніколи не була стандартом ANSI чи ISO.

Windows-1252 та ASCII

Перша частина Windows-1252 (номера об’єктів від 0 до 127) є оригінальним набором символів ASCII. Він містить цифри, великі та малі англійські літери та деякі спеціальні символи.

Для ближчого ознайомлення ознайомтеся з нашим Повним довідником ASCII.

Символи управління

Символи управління (діапазон 00-31 плюс 127) призначені для керування апаратними пристроями.

Символи управління (крім горизонтальної табуляції, переходу рядка та повернення каретки) не мають нічого спільного з документом HTML.
