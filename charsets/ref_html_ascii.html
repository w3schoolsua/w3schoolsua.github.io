ASCII був першим набором символів (стандартом кодування), який використовувався між комп’ютерами в Інтернеті.

І ISO-8859-1 (за замовчуванням у HTML 4.01) і UTF-8 (за замовчуванням у HTML5) створено на основі ASCII.

Набір символів ASCII

ASCII означає "American Standard Code for Information Interchange" / "Американський стандартний код для обміну інформацією".

Він був розроблений на початку 60-х як стандартний набір символів для комп’ютерів та електронних пристроїв.

ASCII — це 7-розрядний набір символів, що містить 128 символів.

Він містить цифри від 0 до 9, великі та малі англійські літери від A до Z, а також деякі спеціальні символи.

Набори символів, які використовуються в сучасних комп’ютерах, у HTML та в Інтернеті, базуються на ASCII.

У наступних таблицях наведено 128 символів ASCII та їх еквівалентну кількість.

ASCII Символи керування пристроєм

Керуючі символи ASCII (діапазон 00-31 плюс 127) розроблено для керування апаратними пристроями.

Символи керування (крім горизонтальної табуляції, переводу рядка та повернення каретки) не мають нічого спільного з документом HTML.
