UTF-8 C0 Елементи керування та базова латиниця

❮ Prev Next ❯

Діапазон: десятковий 0-127. Шістнадцятковий 0000-007F..

Набір символів такий самий, як оригінальний набір символів ASCII.

Якщо ви хочете, щоб спеціальні символи відображалися в HTML, ви можете використати об’єкт HTML, наведений у таблиці нижче.

Якщо символ не має сутності HTML, ви можете використовувати десяткове (dec) або шістнадцяткове (hex) посилання.

Приклад

<p>My name is Johnny &quot;Bang&quot; Johnson</p>

<p>My name is Johnny &#34;Bang&#34; Johnson</p>

<p>My name is Johnny &#x0022;Bang&#x0022; Johnson</p>

Відображатиметься як:

My name is Johnny "Bang" Johnson

My name is Johnny "Bang" Johnson

My name is Johnny "Bang" Johnson
