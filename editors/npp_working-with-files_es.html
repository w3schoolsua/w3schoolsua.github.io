Trabajando con archivos

Un "archivo" es la unidad básica de lo que se edita en Notepad++, pero ese término en realidad abarca varios conceptos relacionados. Principalmente, el "archivo" es la serie de bytes almacenados en un disco u otro medio de almacenamiento y al que se accede a través del sistema de archivos del ordenador. De forma pedante, si el documento que se está editando nunca se ha guardado en el sistema de archivos, técnicamente no es un archivo, aunque el uso común también aplica ese término a documentos no guardados. El "documento" se refiere al texto que se está editando, ya sea un documento nuevo sin guardar o un documento previamente guardado como archivo en el sistema de archivos. Finalmente, en Notepad++, cada documento se presenta en una pestaña en una de las dos Vistas de la interfaz de usuario de Notepad++, que son los contenedores gráficos que Notepad++ utiliza para manipular archivos y otros documentos, aunque muchos usuarios también consideran el elemento de la interfaz de usuario como el "archivo".

File Menu / Menú Archivo

El menú Archivo contiene muchas de las acciones habituales de operación con archivos que se encuentran en la mayoría de las aplicaciones. Notepad++ también incluye otras acciones personalizadas que son útiles para comprender.

  • New: Crea un nuevo documento y muestra ese documento vacío en una pestaña en la vista activa.
  • Open: Abre un archivo existente del sistema de archivos y muestra ese documento en una pestaña en la vista activa.
  • Abrir carpeta contenedora >:
    • Explorer: Abre una instancia del Explorador de Windows, comenzando en la carpeta (también conocida como “directorio”) que contiene el archivo activo.
    • cmd: Abre una instancia de cmd.exe de Windows, comenzando en la carpeta que contiene el archivo activo.
    • Folder as Workspace: Abre la carpeta del archivo activo en el panel Carpeta como espacio de trabajo de Notepad++.
  • Open in Default Viewer: Esto utilizará la “asociación de tipo de archivo” de Windows para el archivo activo (según la extensión del archivo).
    • Equivale a hacer doble clic en un archivo en el Explorador de Windows.
    • Si el documento nunca se ha guardado como archivo, esta acción aparecerá inactiva y no se podrá usar la entrada del menú.
    • Si el tipo de archivo no tiene una acción predeterminada asociada en Windows, esta acción aparecerá inactiva y no se podrá usar la entrada del menú.
    • Si el tipo de archivo está asociado con Notepad++ y, por lo tanto, simplemente se reabriría el archivo en Notepad++, esta acción aparecerá inactiva y no se podrá usar la entrada del menú.
  • Abrir carpeta como espacio de trabajo: Abre la carpeta del archivo activo en el panel Carpeta como espacio de trabajo de Notepad++.
  • Recargar desde el disco: Esto releerá los bytes del archivo desde el sistema de archivos, reemplazando el contenido actual del documento que se está editando con la última versión guardada (y, por lo tanto, perdiendo cualquier cambio que se haya realizado en Notepad++).
    • Si tiene activada la opción Configuración > Preferencias > MISC > Detección automática del estado de archivo, Notepad++ puede ejecutar automáticamente el equivalente de este comando, sin que usted tenga que usar esta entrada de menú.
    • Se borrarán todos los marcadores o estilos de token que hayas configurado previamente.
    • Se restablecerá cualquier historial de cambios registrado para el archivo y el archivo aparecerá sin colores de margen del historial de cambios.
  • Save: Guarda el documento actual en el sistema de archivos utilizando el mismo nombre de archivo.
    • Si el documento no tiene ningún archivo asociado, esta acción aparecerá inactiva y no se podrá usar la entrada del menú. En su lugar, utilice Archivo > Guardar como.
  • Save As: Guarda el documento actual en el sistema de archivos y solicita el nuevo nombre de archivo a utilizar.
    • Si el documento se había guardado previamente en un archivo, esto creará una nueva copia del archivo y Notepad++ continuará trabajando con el documento asociado con el nuevo archivo, dejando el archivo antiguo con el contenido que se guardó por última vez.
  • Save a Copy As: Guarda el documento actual en el sistema de archivos y solicita el nuevo nombre de archivo a utilizar.
    • El nuevo archivo se guardará, pero Notepad++ seguirá trabajando con el documento asociado al nombre de archivo original. Por lo tanto, esta acción crea una copia de seguridad del archivo, pero permite seguir trabajando con el archivo anterior, mientras que "Guardar como" usa el original como copia de seguridad y permite seguir trabajando con el nuevo archivo.
    • Normalmente, durante la ejecución de este comando, el nuevo archivo creado no se abre en una pestaña de edición de Notepad++. Para abrir el nuevo archivo durante el proceso, mantenga presionada la tecla Mayús mientras pulsa el botón Guardar en el cuadro de diálogo Guardar como.
  • Save All: Guarda todos los documentos que están abiertos actualmente en cualquiera de las dos vistas de Notepad++.
  • Rename: Solicita un nuevo nombre para el documento actual.
    • Si el documento está asociado a un archivo, Notepad++ abrirá un cuadro de diálogo estándar de Windows para guardar el archivo con un nuevo nombre en el sistema de archivos, y el nombre de archivo antiguo ya no existirá.
    • Si el documento es nuevo y no se ha guardado, se cambiará el nombre que se muestra en la pestaña del documento actual. Si posteriormente se selecciona "Guardar como", se usará como nombre sugerido predeterminado en el cuadro de diálogo "Guardar como".
  • Close: Cierra el documento activo y pregunta si se ha modificado desde que se abrió o creó..
  • Close All: Cierra todos los documentos que están abiertos en la instancia actual de Notepad++ y pregunta si se debe guardar si uno o más archivos se han modificado desde que se abrieron o crearon.
    • En el cuadro de diálogo emergente, Sí guardará el archivo, No lo cerrará sin guardar (perderá todos los cambios) y Cancelar evitará que Notepad++ cierre todos los archivos.
    • Si se le solicitan varios archivos, también obtendrá Sí a todo para guardar todos los archivos modificados y No a todo para perder los cambios de todos los archivos modificados.
  • Cerrar varios documentos >:
    • Cerrar todos excepto el documento activo: Cierra todos los documentos abiertos, excepto el documento que se está editando activamente.
    • Cerrar todos excepto los documentos anclados: Cierra todos los documentos abiertos, a menos que su pestaña esté anclada.
    • Cerrar todos a la izquierda: Cierra todos los documentos abiertos que estén a la izquierda del documento activo en la barra de pestañas de la vista actual.
    • Cerrar todos a la derecha: Cierra todos los documentos abiertos que estén a la derecha del documento activo en la barra de pestañas de la vista actual.
    • Cerrar todos sin cambios: Cierra todos los documentos abiertos que no se hayan editado desde la última vez que se abrieron o crearon.
  • Mover a la Papelera de Reciclaje: Cierra el documento activo y mueve el archivo subyacente a la Papelera de Reciclaje de Windows. Si el documento no tiene ningún archivo subyacente, esta acción aparecerá inactiva y no se podrá usar la entrada del menú.
  • Cargar Sesión…: Carga todos los archivos de un archivo de Sesión guardado.
  • Guardar Sesión…: Guarda la sesión actual (lista de archivos abiertos) en un archivo de Sesión.
  • Imprimir: Abre un cuadro de diálogo de impresión para imprimir el archivo, como se describe en la sección "Imprimir".
  • Imprimir Ahora: Imprime el archivo activo sin mostrar el cuadro de diálogo de impresión, utilizando la impresora predeterminada y otras configuraciones predeterminadas del cuadro de diálogo de impresión.
  • Archivos Recientes >: La lista de archivos cerrados recientemente que Notepad++ ha editado.
    • Si [Configuración > Preferencias > Historial de archivos recientes] está configurado para usar un submenú, los archivos recientes estarán en ese submenú; de lo contrario, estarán en el menú principal Archivo.
    • Al hacer clic en una entrada de archivo, se abrirá ese archivo.
    • Restaurar archivo cerrado reciente: Abre el archivo cerrado más reciente, que se encuentra al principio de la lista de archivos recientes.
    • Abrir todos los archivos recientes: Abre todos los archivos de la lista.
    • Vaciar lista de archivos recientes: Elimina todos los archivos de la lista de archivos recientes.
  • Exit: Cierra la aplicación Notepad++.
    • Si tiene marcada la opción Configuración > Preferencias > Copia de seguridad ☑ Recordar la sesión actual para el próximo inicio, Notepad++ guardará la lista actual de archivos en la sesión. Los únicos archivos que se recordarán para el próximo inicio de Notepad++ serán los que estén abiertos en Notepad++.
    • Si no tiene marcada la opción Configuración > Preferencias > Copia de seguridad ☐ Habilitar instantánea de sesión y copia de seguridad periódica, Notepad++ le preguntará si desea guardar los archivos modificados, siguiendo las mismas reglas que en la opción "Cerrar todo", descrita anteriormente. Si tiene marcada esta opción, los documentos no guardados también se incluirán en el archivo de sesión, incluso si no tienen un archivo subyacente.

Filtros de tipo de archivo

Para los diálogos "Guardar como" y similares, hay un campo desplegable "Guardar como tipo". Para el diálogo "Abrir" (y similares), hay un campo desplegable de tipo de archivo a la derecha de la entrada "Nombre de archivo". Estos menús desplegables sirven para filtrar la lista de archivos del diálogo según el tipo. La lista de tipos de archivo del filtro incluirá todos los idiomas del menú Idioma y sus extensiones, tal como se define en el Configurador de estilos. A partir de la versión 8.7, el filtro también incluirá los distintos tipos de archivo y extensiones de Idioma definido por el usuario.

En el cuadro de diálogo "Guardar como", la casilla "☐ Añadir extensión" (descrita en Conmutaciones de preferencias en otros cuadros de diálogo) permitirá que el cuadro de diálogo agregue automáticamente la primera extensión de la lista del filtro activo. Si desea una extensión diferente para ese tipo, o si esa opción no está marcada, puede escribir el nombre del archivo con la extensión para definir explícitamente la extensión que desea para el archivo que está guardando. (Por lo general, al filtrar Guardar como para C++ se agregará automáticamente la extensión .cpp; pero si desea que sea .cxx o .h, puede escribirlo como parte de la entrada Nombre de archivo en lugar de dejar que Notepad++ agregue automáticamente la extensión).

Alternativas al menú Archivo

Muchas de las acciones del menú Archivo tienen múltiples formas de acceder a ellas, como se describe en la Interfaz de usuario y en otras secciones correspondientes del Manual de usuario.

For example,

  • File > Open tiene muchas formas de lograr lo mismo:
    • El atajo de teclado predeterminado es Ctrl+N.
    • La barra de herramientas tiene un icono Nuevo.
    • La barra de menú puede tener un ícono + para crear un nuevo archivo.
    • Puede hacer doble clic en el espacio vacío de la barra de pestañas.
    • Puede utilizar el menú de clic derecho del icono de la bandeja del sistema.
  • File > Close también tiene muchas formas de lograr lo mismo:
    • El atajo de teclado predeterminado es Ctrl+W.
    • La barra de herramientas tiene un icono Cerrar.
    • La barra de menú puede tener un ícono X para cerrar el archivo activo.
    • La sección Interfaz de usuario > Pestañas explica tres más:
      • Puede hacer clic en la X del título de una pestaña determinada para cerrarla, si está configurado Configuración > Preferencias > General > Barra de pestañas > Mostrar botón de cierre.
      • Puede hacer doble clic en el título de la pestaña para cerrarla, si está configurado Configuración > Preferencias > General > Barra de pestañas > ☐ Hacer doble clic para cerrar documento.
      • Puede hacer clic con el botón central en el título de la pestaña para cerrarla.
    • Puede utilizar el menú contextual del título de la pestaña.

En general, es redundante explicar todos los métodos alternativos en varias ubicaciones del Manual. Por lo tanto, si busca una forma de realizar una acción desde el menú Archivo en otra ubicación, puede usar la función Buscar en la página de su navegador (generalmente Ctrl+F o similar) para buscar la acción que busca en la página Interfaz de usuario. Estudiar esa página le permitirá familiarizarse con las distintas ubicaciones donde se pueden encontrar dichas funciones.

Impresión

La acción Archivo > Imprimir abrirá un cuadro de diálogo de impresión estándar de Windows, donde podrá seleccionar su impresora y enviar el texto a la impresora seleccionada. Normalmente, se imprimirá todo el documento, pero puede usar el cuadro de diálogo de impresión para seleccionar solo algunas páginas; si tiene una selección activa en el editor, solo se imprimirá el texto seleccionado.

La impresión del texto se ve afectada por la configuración de Configuración > Preferencias > Imprimir, donde puede decidir aspectos como incluir números de línea en la impresión, el ancho de los márgenes, el texto adicional para imprimir en el encabezado y/o pie de página de cada página, y las Opciones de Color determinarán cómo se propagarán los colores de su tema del Configurador de Estilo en la salida impresa. Usted, como usuario, decide qué configuración de impresión se adapta mejor a sus necesidades y se ve mejor en el soporte de impresión que elija. Tenga en cuenta que, con cualquier tema con fondos de color, algunos de esos colores se imprimirán en el fondo de la página impresa a menos que elija una de las Opciones de Color que no imprima el color de fondo. Sin embargo, con temas más oscuros, que usan texto claro sobre fondos oscuros, si no imprime el color de fondo, el texto será muy claro en el papel blanco. Si tiene una impresora que le permite imprimir en PDF, es una buena forma de ver cómo se verá realmente la impresión sin usar papel ni tinta (y si no tiene una impresora PDF disponible, su motor de búsqueda favorito le mostrará muchas impresoras PDF comerciales y gratuitas buscando algo como "impresora PDF de Windows" o "imprimir en PDF").
