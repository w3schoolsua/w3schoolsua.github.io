Что такое Notepad++

Notepad++ — Это текстовый редактор и редактор исходного кода для использования в Microsoft Windows. Он поддерживает около 80 языков программирования с подсветкой синтаксиса и сворачиванием кода. Он позволяет работать с несколькими открытыми файлами в одном окне благодаря интерфейсу редактирования с вкладками. Notepad++ доступен по адресу GPL и распространяется как бесплатное программное обеспечение.

Вы можете посетить веб-сайт Notepad++ по адресу https://notepad-plus-plus.org/.

Скачайте Notepad++

Скачайте последнюю версию Notepad++ с: https://notepad-plus-plus.org/downloads/.

Определите, требуется ли для вашей системы 64-разрядная, 64- или 32-разрядная сборка Notepad++ для ARM в соответствии с вашим компьютером и операционной системой, а затем выберите пакет, который вы хотите загрузить для этой архитектуры. Большинство пользователей используют программу установки (установщик), поскольку это самый простой путь, однако Notepad++ также доступен в "портативном" или "мини-портативном" формате 7z и zip.

Установить Notepad++ с помощью инсталлятора

  1. Скачайте программу установки
  2. Запустите исполняемый двоичный файл и следуйте установке

Установщик, вероятно, потребует административных привилегий, чтобы установить Notepad++ (и позже, чтобы обновить Notepad++ или установить или обновить плагины или что-то другое, требующее записи в каталог установки). Если у вас нет прав администратора, вам нужно либо сообщить установщику использовать место, где у вас есть разрешение на запись (хотя он все равно может запрашивать права администратора), либо вы можете не использовать установщик и вместо этого запустить портативную версию из каталога, где у вас есть разрешение на запись.

По умолчанию программа установки будет использовать %AppData%\Notepad++ для сохранения ваших настроек; но если вы используете установщик для установки в каталог, отличный от "Program Files", вы можете выбрать опцию установки "Не использовать %APPDATA%" вместо этого использовать "локальную конфигурацию".

Инсталлятор также должен установить расширение оболочки, чтобы добавить редактирование с помощью Notepad++ в контекстное меню Windows, нажав правой кнопкой мыши (ваша конкретная версия или перевод может использовать другую фразу для этой записи).

Если вы выполняете управляемую установку или иным образом хотите управлять установщиком из командной строки, установщик имеет несколько параметров командной строки.

Установленную версию Notepad++ можно удалить с помощью стандартного интерфейса Windows "Add/Remove Programs" для удаления установленных программ (или аналогично в зависимости от версии ОС).

Инсталляция Notepad++ из 7z или zip

Эти инструкции позволят вам запускать портативное или мини-портативное устройство (также называемое «минималистическим»), не требуя административных привилегий.

  1. Создайте новую папку в месте, где у вас есть разрешение на запись
  2. Распакуйте содержимое в новую папку
  3. Запустите Notepad++ из новой папки

Для всех портативных версий все (программа, настройки и плагины) сохраняется в одной иерархии папок. По умолчанию портативные версии не изменяют параметры операционной системы (поэтому никаких ассоциаций файлов или пунктов контекстного меню «Редактировать с помощью Notepad++») – если вы хотите сделать это с портативной версией, вам придется настроить ее вручную.

Основные портативные версии 7z или zip содержат все темы, определенные пользователем языка, автозавершения, локализации и плагины по умолчанию и инструмент администрирования плагинов, поставляемых вместе с установленной версией Notepad++.

Минималистское/мини-портативное издание поставляется только со светлой и темной темой по умолчанию, английской локализацией по умолчанию и без плагинов или администратора плагинов. Однако вы можете отдельно загрузить любой из невключенных конфигурационных файлов, которые вам нужны, из основного выходного репозитория Notepad++, и вы все равно можете установить плагины вручную.

Портативные версии Notepad++ можно удалить, удалив каталог, в который они поступили. Если вы настроили вручную ассоциации файлов или записи контекстного меню в ОС, вы сами должны удалить их.

"Программные файлы". Ограничения

Windows особенно следит за тем, что входит в "Program Files", и ограничивает там разрешения, обычно для установки или редактирования там требуются права администратора.

Когда Notepad++ размещается в «Program Files» (через программу установки или распаковки портативной версии), он намеренно не выполняет doLocalConf.xml, который поступает из папки "Do not Use %APPDATA%" настройки или распаковки портативной версии или создание файла вручную, вместо этого будет использовать ваш каталог %AppData%\Notepad++ для сохранения настроек, не смотря на то, что этот файл там есть.
