PHP lernen

PHP ist eine Server-Skriptsprache und ein leistungsstarkes Tool zum Erstellen dynamischer und interaktiver Webseiten.

PHP ist eine weit verbreitete, kostenlose und effiziente Alternative zu Konkurrenten wie Microsoft ASP.

PHP 8 - aktuelle stabile Version*.

Beginnen Sie jetzt mit dem Erlernen von PHP »

*Hinweis. Seit dem 26, November 2020 ist eine neue stabile Version verfügbar PHP 8.

Einfaches Lernen mit "PHP Tryit"

Mit unserem Online-Editor „PHP Tryit“ können Sie den PHP-Code bearbeiten und das Ergebnis per Klick auf eine Schaltfläche anzeigen.

Beispiel

<!DOCTYPE html>
<html>
<body>

<?php
echo "My first PHP script!";
?>

</body>
</html>
