ISO Мовні коди

HTML атрибут lang може використовуватись для об’яви мови вебсторінки або частини вебсторінки. Це має допомогти пошуковим системам і браузерам.

Відповідно до рекомендацій W3C ви маєте оголосити основну мову для кожної вебсторінки за допомогою атрибуту lang всередині тега <html>, як тут:

<html lang='uk'>
...
</html>

В XHTML мова оголошується всередині тега <html> наступним чином:

<html xmlns='http://www.w3.org/1999/xhtml' lang='uk' xml:lang='uk'>
...
</html>

Також ви можете вказати зміст вебсторінки написаний двома мовами одночасно в тегах <title> та <meta name='description' content='text, текст'>. Це можна зробити, використовуючи атрибут lang для вказівки мови. Наприклад, якщо вебсторінка містить зміст, написаний англійською та українською мовами, то ви можете вказати це так:

Приклад

<html lang='en-US,uk-UA'>
  <head>
    <meta charset='utf-8'>
    <title lang='en'>Title</title>
    <title lang='uk'>Заголовок</title>
    <meta name='description' content='text' lang='en'>
    <meta name='description' content='текст' lang='uk'>
  </head>
  <body>
    <h1>Зміст веб-сторінки написано двома мовами</h1>
    <p>This text is written in English.</p>
    <p>Цей текст написано українською мовою.</p>
  </body>
</html>

ISO 639-1 Коди мови

ISO 639-1 визначає скорочення для мов.

В HTML та XHTML вони можуть використовуватись в атрибутах lang та xml:lang.
