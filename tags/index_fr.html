How many HTML tags are there?

Many beginners who are just starting to learn web technologies often ask: How many tags are there in the HTML5 specification? As can be seen from the above table, as of the end of 2023 in the specification HTML5 is officially approved and supports 107 HTML tags. The 12 tags listed in the table are no longer supported by the HTML5 specification, but are supported by most browsers. Even if you've used these tags before when building your websites, they will still work in all modern web browsers. However, outdated HTML tags are not recommended to be used in new web projects. It is better to follow the W3C recommendations and the officially approved HTML5 specification. In addition, standards are constantly changing. Over time, some tags may disappear, and new tags appear. In order to stay up to date with all the changes that occur in web standards, it is necessary to visit the official web resources where all these changes are published, such as the site of the World Wide Web Consortium W3C: https://www.w3.org/ and WHATWG.
