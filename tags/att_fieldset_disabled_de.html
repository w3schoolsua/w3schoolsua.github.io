HTML <fieldset> disabled Attribut

Beispiel

Deaktivieren Sie eine Gruppe verwandter Formelemente:

<form action="/action_page.html">
  <fieldset disabled>
    <legend>Personalia:</legend>
    <label for="fname">First name:</label>
    <input type="text" id="fname" name="fname"><br><br>
    <label for="lname">Last name:</label>
    <input type="text" id="lname" name="lname"><br><br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br><br>
    <label for="birthday">Birthday:</label>
    <input type="date" id="birthday" name="birthday"><br><br>
    <input type="submit" value="Submit">
  </fieldset>
</form>
Try it Yourself »

Definition und Verwendung

Das Attribut disabled ist ein Boolean-Attribut.

Wenn vorhanden, gibt es an, dass eine Gruppe verwandter Formularelemente (ein Feldset) deaktiviert werden soll.

Ein deaktiviertes Feldset ist unbrauchbar und nicht anklickbar.

Das Attribut disabled kann so gesetzt werden, dass ein Benutzer die Felder erst verwenden kann, wenn eine andere Bedingung erfüllt ist (z. B. das Aktivieren eines Kontrollkästchens). Anschließend kann JavaScript den deaktivierten Wert entfernen und das Feldset wieder nutzbar machen.
