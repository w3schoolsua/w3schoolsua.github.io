Etiqueta HTML <dialog>

❮ Prev Referencia completa de HTML Next ❯

Ejemplo

Usando el elemento <dialog>:

<dialog open>Esta es una ventana de diálogo abierta</dialog>
Try it Yourself »

Definición y uso

La etiqueta <dialog> define un cuadro de diálogo o una subventana.

El elemento <dialog> facilita la creación de cuadros de diálogo emergentes y modales en una página web.

Compatibilidad con navegadores

Los números en la tabla especifican la primera versión del navegador que admite completamente el elemento.

Elemento
<dialog> 37.0 79.0 53.0* No compatible 24.0

* No es compatible de forma predeterminada, pero se puede habilitar en Action (config (set dom.dialog_element.enabled to true).

Atributos

Atributo Valor Descripción
open open Especifique que el elemento de diálogo esté activo y que el usuario pueda interactuar con él

Atributos globales

La etiqueta <dialog> también admite los Atributos globales en HTML.

La etiqueta <dialog> también admite los Atributos de evento en HTML.
