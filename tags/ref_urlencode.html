HTML URL Кодування. Довідник

Кодування URL перетворює символи в формат, який може бути переданий через Інтернет.

URL (Uniform Resource Locator) - Уніфікований вказівник інформаційного ресурсу

Веб-браузери запрошують сторінки з вебсерверів за допомогою URL-адреси.

URL-адреса - це адреса вебсторінки в Інтернет, наприклад: https://www.w3schools.com.

Кодування URL (кодування з процентами)

URL-адреси можна відправляти лише через Інтернет за допомогою набору символів ASCII.

Оскільки URL-адреси часто містять символи поза набором ASCII, URL-адреса має бути перетворена в правильний формат ASCII.

Кодування URL заміняє небезпечні символи ASCII на '%', за якими слідують дві шістнадцяткові цифри.

URL-адреси не можуть містити пробіли. Кодування URL зазвичай замінює пробіл знаком плюс (+) або %20.

Спробуйте самі

Якщо натиснути кнопку Submit ("Надіслати") нижче, браузер закодує вхідну URL-адресу перед тим, як він буде відправлений на сервер. Сторінка на сервері відобразить отримані дані.

Спробуйте ввести інші дані й знову натисніть "Submit" ("Надіслати")

Функції кодування URL-адрес

В JavaScript, PHP та ASP існують функції, які можна використовувати для URL-кодування рядка.

PHP має функцію rawurlencode(), а ASP має функцію Server.URLEncode().

В JavaScript можна використовувати функцію encodeURIComponent().

Натисніть кнопку 'URL Encode', щоб побачити, як функція JavaScript кодує текст.

Примітка: Функція JavaScript кодує пробіл як %20.
