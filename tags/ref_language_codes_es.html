Códigos de idioma ISO

Siempre debes incluir el atributo lang dentro del etiqueta <html>, para declarar el idioma de la página Web. Esto está destinado a ayudar a los motores de búsqueda y navegadores:

<html lang="en">
...
</html>

En XHTML, el idioma se declara dentro de la etiqueta <html> de la siguiente manera:

<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">
...
</html>
