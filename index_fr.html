HTML

Langage de création de pages Web

HTML - langage de balisage hypertexte pour la création de pages Web. Le HTML n'est pas un langage de programmation. À l'aide de balises html, le balisage est effectué sur une page Web, à l'aide de laquelle sont placés divers objets d'un document Web : en-têtes, texte, images, tableaux, listes, cadres, lecteurs vidéo, etc.

Apprendre HTML HTML Référence

HTML Exemple:

<!DOCTYPE html>
<html>
<head>
  <title>HTML Tutorial</title>
<head>
<body>
  <h1>This is a heading</h1>
  <p>This is a paragraph.</p>
</body>
</html>

CSS

Un langage pour styliser les pages Web

CSS (cascading style sheets) — un langage responsable de la présentation visuelle des informations et de la conception d'une page Web. Les styles sont utilisés pour formater le texte et une autre stylisation (conception) des objets sur une page Web. Le CSS est généralement associé au HTML.

JavaScript

Langage de programmation de pages Web

JavaScript (JS) - Le langage de programmation, le troisième composant (avec HTML et CSS) des technologies Web standards, est utilisé pour créer des sites Web. JavaScript rend les pages Web statiques dynamiques, fluides et conviviales. Le langage de programmation JavaScript gère les médias, anime les images, modifie le texte d'une page Web, et bien plus encore.

Apprendre JAVASCRIPT JS Référence

Python

Langage de programmation populaire

Python - un langage de programmation généraliste de haut niveau axé sur l'amélioration de la productivité des développeurs et de la lecture du code. Il est utilisé pour la programmation d'applications générales et pour la programmation Web (développement de sites Web). Recommandé pour les débutants qui apprennent la programmation Web, ainsi que le premier langage de programmation pour débutants en raison de sa simplicité et de son accessibilité.

PHP

Langue du serveur Web

PHP (Hypertext Preprocessor) est un langage de script à usage général utilisé pour développer des applications Web. Il est pris en charge par la grande majorité des hébergeurs et constitue l’un des principaux langages utilisés pour créer des sites Web dynamiques. PHP est souvent utilisé pour créer un CMS - un système de gestion de contenu (moteur de site). PHP s'exécute côté serveur (backend) offrant une interaction avec l'utilisateur. Des sites comme Facebook, Yahoo, Wikipédia, le moteur WordPress, etc. ont été créés en PHP. Le langage de programmation PHP que vous pouvez Apprendre en ligne sur le site W3Schools en français.

C

Langage de programmation

C - est un langage de programmation à usage général utilisé pour créer des systèmes d'exploitation, des systèmes embarqués, des pilotes, des compilateurs et d'autres logiciels système. Il est également largement utilisé pour le développement d’applications dans le domaine des systèmes embarqués, du calcul scientifique et de la mise en œuvre de logiciels.

C#

Langage de programmation

C# - Ce langage de programmation est un langage orienté objet développé par Microsoft. Il est utilisé pour développer diverses applications, notamment des applications pour les systèmes d'exploitation Windows, des applications Web, des jeux, des applications mobiles et bien plus encore. C# est également utilisé pour développer des logiciels sur la plateforme .NET.

Apprendre C#

Go - Ce langage de programmation, souvent appelé Golang, est un langage hautes performances développé chez Google. Il est utilisé pour créer des programmes fiables, efficaces et rapides.
