W3Schools Онлайн редактор коду

За допомогою онлайн-редактора коду W3Schools ви можете редагувати код HTML, CSS та JavaScript та переглядати результат у своєму браузері.


Виконати » Розмір результату:

Приклад

<!DOCTYPE html>
<html>
<head>
<title>Заголовок сторінки</title>
<style>
body {
  background-color: black;
  text-align: center;
  color: white;
}
</style>
</head>
<body>

<h1>Це заголовок</h1>
<p>Це параграф.</p>
<img src="../images/avatar.png" alt="Аватар" style="width:200px">

</body>
</html>

Пояснення роботи онлайн веб-редактора

Вікно ліворуч доступне для редагування - відредагуйте код і клікніть кнопку "Виконати", щоб переглянути результат в правому вікні (відображення в браузері).

"Розмір результату" повертає ширину і висоту вікна результатів в пікселях (навіть при зміні розміру вікна браузера).

Ви можете контролувати розмір вікна за допомогою смуги між вікнами (жолоб, що перетягується).

Іконки пояснюються в таблиці нижче:
