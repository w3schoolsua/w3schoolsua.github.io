Теорія

Головна мета IntelliJ IDEA — підвищити вашу продуктивність під час написання коду. Ми підготували кілька основних порад, щоб підвищити швидкість і комфорт вашої розробки. Це для типових випадків, але ви також можете знайти більше на Вебсторінці IntelliJ IDEA.

§1. Використовуйте комбінації клавіш

Ми заохочуємо вас використовувати комбінації клавіш, оскільки вони можуть значно пришвидшити ваше кодування та навіть зменшити ризики повторних навантажень. IntelliJ IDEA — це IDE, орієнтована на клавіатуру. Ви можете вибрати одну з попередньо налаштованих схем швидкого доступу або розкладку клавіш у File | Settings | Keymap:

Ви завжди можете детальніше ознайомитися зі списком дій і відповідних ярликів, доступних через пошук:

Або ви можете шукати дію за ярликом:

Ви також можете налаштувати власну розкладку клавіш, якщо вам потрібен настроюваний список ярликів.

Примітка. Якщо ви використовуєте розкладку клавіш за замовчуванням у вашій ОС і бажаєте роздрукувати красиву шпаргалку, перейдіть до Help | Keymap Reference для macOS або Help | Keyboard Shortcuts PDF для Windows.

§2. Перейти на темну сторону

IntelliJ IDEA спочатку використовує схему світлих кольорів за замовчуванням, але ви завжди можете переключитися на темну схему Darcula. Ви можете перейти до неї в File | Settings | Appearance & Behaviour | Appearance, вибравши Darcula зі спадного списку Theme:

Примітка: ви можете використовувати ⌃ ` (у macOS) або Ctrl + ` (у Windows і Linux), щоб швидко переключатися між кольоровими схемами та розкладками клавіш.

§3. Локальний термінал

IntelliJ IDEA має локальний термінал, який дає змогу отримати доступ до командного рядка. Залежно від вашої платформи ви можете працювати з командним рядком, PowerShell, Bash тощо. За допомогою терміналу ви можете виконати будь-яку команду, не виходячи з IDE.

Щоб запустити консоль, натисніть ⌥ F12 / Alt + F12 або натисніть кнопку вікна інструмента терміналу:

§4. Висновок

IntelliJ IDEA дозволяє:

  • вибрати схему за допомогою комбінацій клавіш у File | Settings | Keymap і знайти потрібний ярлик за допомогою пошуку
  • змінити тему за допомогою File | Settings | Appearance & Behaviour | Appearance
  • відкрити термінал за допомогою ⌥ F12 / Alt + F12 або натиснути кнопку вікна терміналу

Тепер у вас є солідний досвід використання загальних функцій IntelliJ. Настав час застосувати їх на практиці!

Ви також можете переглянути Урок на hyperskill.org.

Практичні завдання та відповіді

Подаються завдання та варіанти відповідей. Правильний варіант виділено синім кольором.

№1. Схеми швидкого доступу

Питання: Де можна знайти попередньо налаштовані схеми швидкого доступу?

Виберіть один варіант зі списку:

  • у IntelliJ IDEA немає схем швидкого доступу
  • у File | Settings | Shortcuts
  • у File | Settings | Keymap ✔
  • у спеціальній попередньо встановленій пошуковій програмі

Пояснення. Попередньо налаштовані схеми швидкого доступу можна знайти у File | Settings | Keymap у IntelliJ IDEA.

№2. Darcula

Запитання: Що таке Darcula в IntelliJ IDEA?

Виберіть один варіант зі списку:

  • Культура читабельності програми розробників
  • Темна колірна схема для IntelliJ IDEA ✔
  • Шрифт із закривавленими літерами
  • Назва терміналу в IntelliJ IDEA

Пояснення. Darcula — це схема темних кольорів для IntelliJ IDEA.

№3. Знайти ярлики

Питання: Як швидко знайти ярлики потрібних дій в IDEA?

Виберіть один варіант зі списку:

  • знаходження його в Help | Getting started
  • за допомогою інструмента "пошук" у File | Settings | Keymap ✔
  • перевіривши всю інформацію у File | Settings | Keymap
  • знаходження його в Navigate | Shortcuts

Пояснення. Ви можете швидко знайти ярлики потрібних дій у IntelliJ IDEA за допомогою інструмента «пошук» у File | Settings | Keymap.

№4. Комбінації клавіш

Питання: для чого призначені комбінації клавіш?

Виберіть один варіант зі списку:

  • Вони для людей, які не можуть дозволити собі мишку
  • Завдяки їм ви читаєте швидше
  • Вони пришвидшують кодування ✔
  • Вони надають доступ до більш складних функцій

Пояснення. Комбінації клавіш призначені для прискорення кодування.

№5. Командний термінал

Питання: Коли можна отримати доступ до командного терміналу в IntelliJ IDEA?

Виберіть один варіант зі списку:

  • Завжди ✔
  • Під час процесу встановлення
  • У командному режимі
  • Ніколи, вам потрібно встановити його самостійно

Пояснення. Ви завжди можете отримати доступ до командного терміналу в IntelliJ IDEA.

№6. Запустіть термінал IntelliJ IDEA

Питання: Що можна використовувати для запуску локального терміналу IntelliJ IDEA?

Виберіть один або кілька варіантів зі списку:

  • Alt + F / ⌥ F
  • View | Tool Windows | Terminal ✔
  • Alt + F1 / ⌥ F1
  • Alt + F12 / ⌥ F12 ✔

Пояснення. Щоб запустити локальний термінал IntelliJ IDEA, ви можете скористатися наведеними нижче параметрами: View | Tool Windows | Terminal; Alt + F12 / ⌥ F12.

№7. Чому IntelliJ IDEA?

Питання: Чому можна використовувати IntelliJ IDEA?

Виберіть один варіант зі списку:

  • Це єдиний спосіб написати виконуваний код
  • Це модна програма
  • Він попередньо встановлений на кожній ОС
  • Він надає корисні функції для прискорення розробки ✔

Пояснення. Можна використовувати IntelliJ IDEA, оскільки вона надає корисні функції для прискорення розробки.

№8. Змініть тему

Питання: Які швидкі клавіші можна використовувати для швидкого перемикання між колірними схемами та розкладками клавіш?

Виберіть один або кілька варіантів зі списку:

  • ⌥ + F12
  • Ctrl + ` ✔
  • ^ + ` ✔
  • Alt + F12

Пояснення. Для швидкого перемикання між кольоровими схемами та розкладками клавіш можна використовувати наступні комбінації клавіш: Ctrl + `; ^ + `.

Як користуватися основними функціями IntelliJ IDEA?

IntelliJ IDEA — це інтегроване середовище розробки (IDE), яке надає широкий спектр функцій, які допомагають розробникам ефективно писати код. Деякі з його основних функцій включають:

  1. Редагування коду: IntelliJ IDEA надає редактор коду з такими функціями, як підсвічування синтаксису, доповнення коду та інструменти рефакторингу. Це полегшує розробникам написання, читання та змінення коду.
  2. Налагодження: IntelliJ IDEA має вбудований налагоджувач, який дозволяє розробникам виявляти та виправляти помилки у своєму коді. Він надає такі функції, як точки зупину, спостереження та трасування стека, щоб допомогти розробникам усунути проблеми.
  3. Контроль версій: IntelliJ IDEA інтегрується з такими системами контролю версій, як Git, Subversion і Mercurial. Це полегшує розробникам керування змінами коду та співпрацю з іншими.
  4. Інструменти збірки: IntelliJ IDEA підтримує такі інструменти збирання, як Gradle і Maven. Це дозволяє розробникам автоматизувати процес збирання та ефективніше керувати залежностями.
  5. Тестування: IntelliJ IDEA надає інструменти для тестування коду, включно з модульним тестуванням і інтеграційними платформами тестування. Це допомагає розробникам переконатися, що їхній код працює правильно.

Щоб використовувати ці функції в IntelliJ IDEA, розробники можуть використовувати різні меню, панелі інструментів і комбінації клавіш, надані IDE. Вони також можуть налаштувати ці параметри відповідно до свого робочого процесу та вподобань.
