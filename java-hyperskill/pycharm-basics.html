Теорія

Коли ви починаєте програмувати, Інтегроване середовище розробки (IDE) може допомогти з основними робочими процесами розробника, дозволяючи вам продовжувати зосереджуватися на особливостях мови. Радимо вам використовувати PyCharm для своїх проєктів на Python. PyCharm – це спеціальна IDE Python, яка надає різноманітні корисні функції, як-от доповнення коду, перевірка коду, навігація проєктом і рефакторинг.

§1. Огляд

PyCharm доступний у двох версіях: Community та Professional. Видання Community є безкоштовним і з відкритим кодом. Він постачається з усім необхідним для розумної та інтелектуальної розробки Python, включаючи допомогу в коді, рефакторинг, візуальне налагодження та інтеграцію контролю версій. PyCharm Professional — це платна версія PyCharm, призначена для професійної розробки Python, Інтернету та науки про дані. За допомогою PyCharm Professional ви можете створювати веб-фреймворки, розробляти віддалено, аналізувати великі дані та працювати з блокнотами Jupyter. Додаткову інформацію про кожен випуск див. на сторінці продукту PyCharm.

§2. Встановлення та запуск PyCharm

Перш ніж почати, перевірте вимоги до пам’яті та операційних систем на сторінці Системні вимоги PyCharm.

Встановити Python

PyCharm — це специфічне середовище розробки Python, тому вам знадобиться Python, щоб почати роботу. Завантажте його з сайту python.org, а потім інсталюйте відповідно до вимог вашої операційної системи.

Завантажити та встановити PyCharm

  1. Завантажте PyCharm із https://www.jetbrains.com/pycharm/download.
  2. Запустіть інсталятор і дотримуйтеся вказівок майстра. Додаткову інформацію див. у Посібнику зі встановлення PyCharm.

Зауважте, що ви можете вибрати окрему установку або додаток Toolbox, що корисно, коли вам потрібно встановити кілька JetBrains IDE або працювати з кількома версіями PyCharm.

Запустити PyCharm

Щоб запустити PyCharm, виконайте такі дії залежно від вашої операційної системи:

  • Windows: запустіть PyCharm за допомогою меню «Пуск» на Windows або ярлика на робочому столі
  • macOS: запустіть програму PyCharm із каталогу Applications
  • Linux: запустіть сценарій оболонки pycharm.sh у каталозі встановлення під bin

Екран привітання

Після запуску PyCharm ви побачите екран привітання:

На екрані привітання ви можете знайти всі параметри та налаштування, щоб почати роботу в PyCharm. Найшвидший спосіб — створити новий проєкт з нуля або відкрити існуючий проєкт із локального диска чи репозиторію.

Початкова сторінка також містить посилання на початковий огляд PyCharm, який допоможе вам ознайомитися з основним робочим процесом розробника всього за 7 хвилин. Перегляньте екскурсію, щоб почати ще легше.

Швидкий старт

Ви можете почати роботу в середовищі IDE, не встановлюючи нічого попередньо, оскільки PyCharm надає майже всі функції готові. Тим не менш, що б ви не робили в PyCharm, ви робите в контексті проєкту. Проєкт — це організаційна одиниця, яка служить основою для допомоги в кодуванні, масового рефакторингу та узгодженості стилю кодування. Отже, вам потрібно почати з налаштування проєкту. У вас є такі варіанти:

  • Створити новий проєкт — створіть новий проєкт у своїй файловій системі та налаштуйте середовище Python за допомогою інсталяції Python.
  • Відкрити існуючий проєкт — відкрити існуючий проєкт PyCharm. Ви також можете відкрити будь-яку папку у своїй файловій системі, і PyCharm виявить будь-яке попередньо налаштоване середовище Python.
  • Отримати проєкт із системи контролю версій — клонувати проєкт зі сховища Git, Mercurial або Subversion.

Індивідуальний старт

На екрані привітання можна налаштувати стандартні параметри IDE. Натисніть Customize (Налаштувати) та виберіть іншу колірну тему або встановіть прапорець Sync with OS (Синхронізувати з ОС), щоб використовувати тему вашої системи за умовчанням. Тут також можна налаштувати параметри доступності або вибрати іншу схему клавіш.

Ви також можете натиснути All settings (Усі налаштування), щоб відкрити діалогове вікно налаштувань. Параметри, які ви зараз змінюєте, стануть новою конфігурацією за замовчуванням для ваших проєктів та IDE.

Хоча PyCharm Community містить усі основні функції, ви можете розширити їх, установивши плагіни. Натисніть Plugins (Плагіни) на лівій панелі екрана привітання, а потім завантажте та встановіть необхідні плагіни зі Сховище плагінів PyCharm.

Тепер ви готові створити свою першу програму Python.

§3. Створити проєкт Python у PyCharm

Щоб створити новий проєкт Python, виконайте такі дії:

  1. На екрані привітання виберіть New Project (Новий проєкт):
  2. У діалоговому вікні New Project PyCharm заповнює всі необхідні поля за вас. IDE створить віртуальне середовище Python, ізольований «квазі» дистрибутив Python із доступними бібліотеками та вбудований менеджер пакетів Python, який дозволяє встановлювати зовнішні бібліотеки. Список інтерпретатора Base вказує шлях до фактичного Python, який ви завантажуєте та інсталюєте на своєму комп’ютері.

    3. Тож на цьому етапі ви можете вибрати один тип середовища: віртуальне середовище (за замовчуванням), Conda, Pipenv або Poetry та створити його за допомогою одного з інтерпретаторів Python, установлених у вашій системі. Зверніться до Creating Python Projects (Створення проєктів Python), щоб дізнатися більше про інші параметри проєкту.

  3. Зверніть увагу на прапорець Create a main.py welcome script (Створити сценарій привітання main.py). Він вибраний за замовчуванням. Якщо цей параметр увімкнено, PyCharm створює файл Python із базовим кодом, який може стати гарною відправною точкою для вашої програми.
  4. Натисніть Create (Створити), щоб завершити завдання.

Після створення проєкту файл main.py відкривається в редакторі. Цей файл показує базовий скрипт і містить корисні підказки, щоб ви могли почати редагувати код у PyCharm. А поки пригадайте одну з найбільш корисних комбінацій клавіш Shift + Shift. Вона відкриває діалогове вікно, де можна знайти будь-яку команду, параметр, конструкцію коду або файл проєкту. Зверніться до розділу Робота з вихідним кодом у PyCharm, щоб дізнатися більше про доступну допомогу з кодування.

Ви можете додати більше файлів різних типів до свого проєкту, а також каталогів і пакетів Python. Усі вони перераховані у вікні інструментів проєкту. Щоб дізнатися більше про додавання файлів і каталогів до проєкту, зверніться до наступної теми веб-довідки PyCharm: Заповнення проєктів PyCharm.

Ви вже дізналися про редактор PyCharm і вікно інструменту Project. Давайте розглянемо весь інтерфейс користувача PyCharm:

Основними елементами інтерфейсу IDE є:

  1. Головне меню. Його позиція залежить від ОС;
  2. Панель навігації та головна панель інструментів;
  3. Вікно інструментів проєкту;
  4. Редактор;
  5. Вікна інструментів, які включають консоль Python і вікна пакетів Python;
  6. Селектор інтерпретатора Python. Він показує середовище, яке зараз налаштовано для проєкту.

Після цього ви створили проєкт PyCharm і налаштували для нього інтерпретатор Python. Ви можете створити більше інтерпретаторів Python, якщо вам потрібні різні середовища для запуску ваших скриптів (наприклад, якщо ви хочете виконати їх на різних версіях Python).

§4. Створення інтерпретаторів Python

Найпростіший спосіб відкрити налаштування інтерпретатора Python — натиснути селектор Python Interpreter (Інтерпретатор Python), розташований у нижній частині вікна PyCharm, у рядку стану.

  1. Натисніть засіб вибору інтерпретатора. Ви повинні бачити всі інтерпретатори, налаштовані для використання в усіх проєктах.

  2. Виберіть у меню Add New Interpreter (Додати новий інтерпретатор > Add Local Interpreter (Додати локальний інтерпретатор).
  3. Ви можете побачити різні типи інтерпретаторів у діалоговому вікні Add Interpreter (Додати інтерпретатор):
  4. Ви можете створити новий інтерпретатор або продовжити використовувати існуючий.

    5. Для нового інтерпретатора виберіть Базовий інтерпретатор зі списку або натисніть "..." і знайдіть виконуваний файл Python у своєму файлова система. Для наявного інтерпретатора виберіть Інтерпретатор або подібним чином натисніть "...", щоб знайти виконуваний файл Python.

Якщо PyCharm не може виявити Python на вашій машині, він пропонує два варіанти: завантажити найновіші версії Python з python.org або вкажіть шлях до виконуваного файлу Python (у разі нестандартної інсталяції). Додаткову інформацію та відповідні процедури див. у розділі Налаштування інтерпретатора Python у PyCharm.

Коли ви зберігаєте зміни, новостворений інтерпретатор налаштовується для поточного проєкту.

Ключовим моментом для створення різних ізольованих середовищ є збереження різних наборів пакетів Python. Отже, давайте навчимося встановлювати пакети в PyCharm.

§5. Встановіть пакети Python

Вікно інструменту Python Packages (Пакети Python) забезпечує найшвидший і акуратний спосіб попереднього перегляду та встановлення пакетів для вибраного інтерпретатора Python.

Щоб установити пакет зі сховища за умовчанням:

  1. Введіть назву пакета в поле Search (Пошук) у вікні інструмента пакетів Python.
  2. Знайдіть пакет у списку типового сховища
  3. Натисніть кнопку Install (Встановити) у верхньому правому куті вікна інструментів

Щоб дізнатися більше про встановлення пакетів Python, див. Встановлення, видалення та оновлення пакетів у PyCharm.

§6. Висновок

Підсумовуючи:

  • PyCharm — це IDE для Python. Він має дві версії: Community (безкоштовна) і Professional (30-денна безкоштовна пробна версія).
  • Щоб установити PyCharm, завантажте програму встановлення з https://www.jetbrains.com /pycharm/download.
  • На екрані привітання можна швидко створити, відкрити чи клонувати проєкт Python.
  • Ви можете налаштувати IDE за допомогою налаштувань PyCharm.
  • Щоб розпочати роботу з PyCharm, вам потрібно створити проєкт, налаштувати віртуальне середовище Python (інтерпретатор Python) і додати свій код Python до main.py або до інших файлів Python.
  • PyCharm дозволяє налаштовувати різні інтерпретатори Python залежно від вашої операційної системи та версії PyCharm.
  • За допомогою вікна інструментів пакетів Python ви можете швидко інсталювати пакети на вибраному інтерпретаторі Python.
  • Детальнішу інформацію та конкретні процедури можна знайти в веб-довідці PyCharm.

Ви також можете переглянути Урок на hyperskill.org.
