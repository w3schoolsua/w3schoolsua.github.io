Приклади у кожному розділі

Наш онлайн-редактор "Спробуйте самі" полегшує вивчення мови Go. Ви можете редагувати код мови програмування Go та переглядати результат у своєму браузері.

Приклад

package main
import ("fmt")

func main() {
  fmt.Println("Hello World!")
}
Спробуйте самі »

Клікніть на кнопку "Спробуйте самі", щоб побачити, як вона працює.

Рекомендується використовувати меню уроків

Якщо ви новачок і тільки-но починаєте вивчати мову програмування Go, то рекомендується вивчати цей підручник з мови Go у послідовності тем, зазначених у меню.

Якщо у вас великий екран (наприклад, монітор комп’ютера), меню завжди буде ліворуч.

Якщо у вас маленький екран (наприклад, якщо ви користуєтесь смартфоном або планшетом), відкрийте меню, натиснувши іконку верхнього меню ☰.
