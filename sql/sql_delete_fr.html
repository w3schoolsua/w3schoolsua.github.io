L'instruction SQL DELETE

L'instruction DELETE est utilisée pour supprimer les enregistrements existants dans une table.

DELETE Syntaxe

DELETE FROM table_name WHERE condition;

Remarque : Soyez prudent lorsque vous supprimez des enregistrements dans une table ! Notez la clause WHERE dans l'instruction DELETE. Cette clause spécifie les enregistrements à supprimer. Si vous omettez cette clause, tous les enregistrements de la table seront supprimés !

SQL DELETE Exemple

L'instruction SQL suivante supprime le client « Alfreds Futterkiste » de la table « Customers » :

Exemple

DELETE FROM Customers WHERE CustomerName='Alfreds Futterkiste';

Supprimer tous les enregistrements

Il est possible de supprimer toutes les lignes d'une table sans supprimer la table elle-même. Cela signifie que la structure, les attributs et les index de la table seront conservés :

DELETE FROM table_name;

The following SQL statement deletes all rows in the "Customers" table, without deleting the table:

Exemple

DELETE FROM Customers;
Try it Yourself »

Delete a Table

To delete the table completely, use the DROP TABLE statement:

Exemple

Remove the Customers table:

DROP TABLE Customers;
