JavaScript Булеві (логічні) значення

Булеві (логічні) значення JavaScript можуть мати одне з двох значень: true або false.

Функція Boolean()

Ви можете скористатися функцією Boolean(), щоб дізнатися, чи вираз є true (істинним):

Приклад

Boolean(10 > 9)
Спробуйте самі »

Або ще простіше:

(10 > 9)
Спробуйте самі »

JavaScript Булеві значення. Методи та властивості

Назва Опис
constructor Повертає функцію, яка створила логічний прототип JavaScript
prototype Дозволяє додавати властивості та методи до булевого (логічного) прототипу
toString() Перетворює булеве (логічне) значення на рядок і повертає результат
valueOf() Повертає примітивне значення булевого (логічного) значення
