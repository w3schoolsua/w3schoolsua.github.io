Was ist CSS?

CSS (Cascading Style Sheets) — ist eine Stylesheet-Sprache, die zum Festlegen der Darstellung und des Stils eines Dokuments verwendet wird, das in einer Auszeichnungssprache wie HTML oder XML (einschließlich XML-Dialekten wie SVG, MathML oder XHTML) geschrieben ist.

CSS ist neben HTML und JavaScript eine Eckpfeilertechnologie des World Wide Web.

CSS-Spezifikationen wurden erstellt und werden weiterentwickelt World Wide Web Consortium - W3C.

CSS hat verschiedene Ebenen und Profile. Die nächste Stufe von CSS baut auf den vorherigen auf und fügt neue Funktionen hinzu oder erweitert bereits vorhandene Funktionen. Die Ebenen werden als CSS1, CSS2 und CSS3 bezeichnet. Profile sind eine Reihe von CSS-Regeln auf einer oder mehreren Ebenen, die für bestimmte Geräte- oder Schnittstellentypen erstellt werden. Beispielsweise gibt es CSS-Profile für Drucker, mobile Geräte usw.

CSS (Kaskaden- oder Blocklayout) ersetzte das tabellarische Layout von Webseiten. Der Hauptvorteil des Blocklayouts ist die Trennung von Seiteninhalt (Daten) und seiner visuellen Darstellung (Design).

Ab 2024 ist die aktuelle Version die CSS3-Spezifikation. Die CSS4-Spezifikation befindet sich seit 2011 in der Entwicklung. CSS4-Module bauen auf CSS3 auf und fügen neue Eigenschaften und Werte hinzu. Alle liegen bislang als Entwürfe vor (Arbeitsentwurf) und wurden zu diesem Zeitpunkt noch nicht offiziell genehmigt.

Die offizielle Seite von CSS, auf der Sie immer aktuelle Informationen zu Änderungen und Neuerungen in CSS finden - https://www.w3.org/TR/CSS/#css.

Was ist CSS? Warum wird es benötigt?

CSS (Cascading Style Sheets) ist eine Stilbeschreibungssprache, die eine wichtige Rolle bei der Erstellung schöner und benutzerfreundlicher Websites spielt.

Warum ist das Erlernen von CSS der Schlüssel zum erfolgreichen Webentwickler?

  • Erscheinungsbild: HTML definiert den Textinhalt, während CSS seine visuelle Struktur, sein Layout und seine Ästhetik definiert. Ohne CSS wären Websites weniger attraktiv und für Benutzer schwer zugänglich. Mit CSS-Kenntnissen können wir ein schönes Design erstellen, Farben, Schriftarten, die Positionierung von Elementen ändern und vieles mehr.
  • Ladegeschwindigkeit: CSS hilft, die Codegröße zu reduzieren, was die Seitenladegeschwindigkeit verbessert. Mit einer einzigen CSS-Regel können Sie Stile auf alle Vorkommen eines bestimmten Tags in einem HTML-Dokument anwenden.
  • Benutzererfahrung: Gut gestaltete Schaltflächen, logisch angeordneter Text und benutzerfreundliches Design machen die Website für Benutzer verständlicher. CSS trägt dazu bei, das Website-Erlebnis zu verbessern.
  • Effiziente Entwicklung: Mit CSS können Sie mit einer einzigen Codezeile Stile auf mehrere Seiten anwenden. Wenn Sie beispielsweise Seiten haben, die dasselbe Format haben, können Sie für alle diese Seiten eine CSS-Datei verwenden. Bei Bedarf kann das Design solcher Seiten einfach geändert werden, indem nur der Inhalt der CSS-Datei geändert wird.
