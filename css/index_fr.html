Qu’est-ce que CSS ?

CSS (Cascading Style Sheets) — est un langage de feuille de style utilisé pour spécifier la présentation et le style d'un document écrit dans un langage de balisage tel que HTML ou XML (y compris les dialectes XML tels que SVG, MathML ou XHTML).

CSS est une technologie fondamentale du World Wide Web, aux côtés de HTML et JavaScript.

Les spécifications CSS ont été créées et sont en cours de développement World Wide Web Consortium - W3C.

CSS a différents niveaux et profils. Le niveau suivant de CSS est construit sur les précédents, en ajoutant de nouvelles fonctionnalités ou en étendant les fonctionnalités déjà existantes. Les niveaux sont notés CSS1, CSS2 et CSS3. Les profils sont un ensemble de règles CSS d'un ou plusieurs niveaux, créées pour certains types d'appareils ou d'interfaces. Par exemple, il existe des profils CSS pour les imprimantes, les appareils mobiles, etc.

Le

CSS (mise en page en cascade ou en bloc) a remplacé la disposition tabulaire des pages Web. Le principal avantage de la mise en page par blocs est la séparation du contenu de la page (données) et de sa présentation visuelle (conception).

Depuis 2024, la version actuelle est la spécification CSS3. La spécification CSS4 est en cours de développement depuis 2011. Les modules CSS4 s'appuient sur CSS3 et ajoutent de nouvelles propriétés et valeurs. Tous existent à l'état de brouillons (working draft) et n'ont pas encore été officiellement approuvés.

Le site officiel de CSS, où vous pouvez toujours trouver des informations à jour sur les changements et les innovations à CSS - https://www.w3.org/TR/CSS/#css.

Qu’est-ce que CSS ? Pourquoi est-ce nécessaire ?

CSS (Cascading Style Sheets) est un langage de description de style qui joue un rôle important dans la création de sites Web beaux et conviviaux.

Pourquoi l'apprentissage du CSS est-il essentiel pour devenir un développeur Web performant ?

  • Apparence : HTML définit le contenu textuel, tandis que CSS définit sa structure visuelle, sa mise en page et son esthétique. Sans CSS, les sites Web seraient moins attrayants et difficiles d’accès pour les utilisateurs. La connaissance de CSS nous permet de créer un beau design, de changer les couleurs, les polices, le positionnement des éléments et bien plus encore.
  • Vitesse de chargement : CSS permet de réduire la taille du code, ce qui améliore la vitesse de chargement des pages. Avec une seule règle CSS, vous pouvez appliquer des styles à toutes les occurrences d'une balise particulière dans un document HTML.
  • Expérience utilisateur : des boutons bien conçus, un texte disposé de manière logique et une conception conviviale rendent le site Web plus compréhensible pour les utilisateurs. CSS contribue à améliorer l'expérience de leur site Web.
  • Développement efficace : avec CSS, vous pouvez appliquer des styles à plusieurs pages avec une seule ligne de code. Par exemple, si vous avez des pages ayant le même format, vous pouvez utiliser un seul fichier CSS pour toutes ces pages. Si nécessaire, la conception de ces pages peut être facilement modifiée en modifiant uniquement le contenu du fichier CSS.
