CSS HEX Couleurs

❮ Prev Next ❯

Une couleur hexadécimale est spécifiée avec : #RRGGBB, où les entiers hexadécimaux RR (red/rouge), GG (green/vert) et BB (blue/bleu) spécifient les composants de la couleur.

Valeur HEX

En CSS, une couleur peut être spécifiée à l'aide d'une valeur hexadécimale sous la forme :

#rrggbb

Où rr (rouge), gg (vert) et bb (bleu) sont des valeurs hexadécimales comprises entre 00 et ff (identiques au nombre décimal 0-255).

Par exemple, #ff0000 s'affiche en rouge, car le rouge est réglé sur sa valeur maximale (ff) et les autres sur sa valeur minimale (00).

Pour afficher le noir, définissez toutes les valeurs sur 00, comme ceci : #000000.

Pour afficher le blanc, définissez toutes les valeurs sur ff, comme ceci : #ffffff.

Essayez de mélanger les valeurs HEX ci-dessous :

 

RED

ff

GREEN

0

BLUE

0

Exemple

#ff0000
#0000ff
#3cb371
#ee82ee
#ffa500
#6a5acd

Valeur hexadécimale à 3 chiffres

Vous verrez parfois un code hexadécimal à 3 chiffres dans la source CSS.

Le code hexadécimal à 3 chiffres est une abréviation de certains codes hexadécimaux à 6 chiffres.

Le code hexadécimal à 3 chiffres se présente sous la forme suivante :

#rgb

Où r, g et b représentent les composantes rouge, verte et bleue, avec des valeurs comprises entre 0 et f.

Le code hexadécimal à 3 chiffres ne peut être utilisé que lorsque les deux valeurs (RR, GG et BB) sont identiques pour chaque composante. Ainsi, si nous avons #ff00cc, cela peut s'écrire comme suit : #f0c.

Voici un exemple :

Exemple

body {
  background-color: #fc9; /* same as #ffcc99 */
}

h1 {
  color: #f0f; /* same as #ff00ff */
}

p {
  color: #b58; /* same as #bb5588 */
}
