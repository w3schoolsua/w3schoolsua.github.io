Text drucken

Aus dem vorherigen Kapitel haben Sie gelernt, dass Sie die Methode println() verwenden können, um Werte auszugeben oder Text in Java zu drucken:

Beispiel

System.out.println("Hello World!");

Try it Yourself »

Sie können beliebig viele println()-Methoden hinzufügen. Beachten Sie, dass für jede Methode eine neue Zeile hinzugefügt wird:

Beispiel

System.out.println("Hello World!");
System.out.println("I am learning Java.");
System.out.println("It is awesome!");

Try it Yourself »

Doppelte Anführungszeichen

Der Text muss in doppelte Anführungszeichen eingeschlossen werden "".

Wenn Sie die doppelten Anführungszeichen vergessen, tritt ein Fehler auf:

Beispiel

System.out.println("This sentence will work!");
System.out.println(This sentence will produce an error);

Try it Yourself »

Die Print()-Methode

Es gibt auch eine print()-Methode, die der ähnelt println().

Der einzige Unterschied besteht darin, dass am Ende der Ausgabe keine neue Zeile eingefügt wird:

Beispiel

System.out.print("Hello World! ");
System.out.print("I will print on the same line.");

Try it Yourself »

Beachten Sie, dass wir zur besseren Lesbarkeit ein zusätzliches Leerzeichen hinzufügen (im Beispiel oben nach „Hello World!“).

In diesem Tutorial verwenden wir nur println(), da es die Codeausgabe leichter lesbar macht.
