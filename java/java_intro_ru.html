Что такое Java?

Java — популярный язык программирования, созданный в 1995 году.

Он принадлежит компании Oracle, и более 3 миллиарда устройств работают на Java.

Используется для:

  • Мобильные приложения (особенно Android)
  • Настольные приложения
  • Веб-приложения
  • Веб-сервера и серверы программ
  • Игры
  • Подключение к базе данных
  • И многое, многое другое!

Зачем использовать Java?

  • Java работает на разных платформах (Windows, Mac, Linux, Raspberry Pi и др.)
  • Это один из самых популярных языков программирования в мире
  • Его легко освоить и использовать
  • У него бесплатный и открытый код
  • Он безопасен, быстрый и мощный
  • Он имеет большую поддержку сообщества (десятки миллионов разработчиков)
  • Java — это объектно-ориентированный язык программирования, который даёт четкую структуру программам и позволяет повторно использовать код, снижая затраты на разработку
  • Поскольку Java близок к C++ и C#, это позволяет программистам на этих языках легко перейти на Java или наоборот

Анекдот о программисте на Java

В программистском коллективе было два программиста, которые постоянно раздражали друг друга. Один из них был программистом на C#, а другой – на языке Java. Они постоянно спорили, какой язык программирования лучше.

Однажды они решили устроить соревнования, чтобы определить, кто из них лучший программист. Соревнование заключалось в написании программы, которая решала поставленную задачу в кратчайшие сроки.

Программист на C# начал разработку и вложил всю свою энергию, используя все преимущества языка. Через час он объявил, что закончил.

Программист на Java, видя это, улыбнулся и сказал: "Ты шутишь?! Я могу написать ту же программу на Java вдвое быстрее тебя!"

Они согласились и начали соревнования. Через 30 минут программист Java объявил, что закончил.

Удивленный программист на C# попросил показать его код. Оказалось, что весь код Java состоял из одной линии, а именно: "System.out.println('Hello World!');"

Программист на C# был шокирован и сказал: "Я написал сотни строк кода, а ты сделал это за одну строчку! Как такое может быть?!."

На это программист на Java улыбнулся и сказал: "Ты не поверишь, но одна строчка моего Java-кода ещё и работает на любой операционной системе!"

На следующий день программист на C# уволился с работы и больше его уже никто не видел...
