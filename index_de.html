HTML

Sprache zum Erstellen von Webseiten

HTML - (hypertext markup language) Hypertext-Markup-Sprache zum Erstellen von Webseiten. HTML ist keine Programmiersprache. Mithilfe von HTML-Tags erfolgt das Markup auf einer Webseite, mit deren Hilfe verschiedene Objekte eines Webdokuments platziert werden: Überschriften, Text, Bilder, Tabellen, Listen, Frames, Videoplayer usw.

Lernen HTML HTML Referenz

HTML Beispiel:

<!DOCTYPE html>
<html>
<head>
  <title>HTML Lernprogramm</title>
<head>
<body>
  <h1>Das ist eine Überschrift</h1>
  <p>Das ist ein Absatz.</p>
</body>
</html>

CSS

Eine Sprache zum Gestalten von Webseiten

CSS (cascading style sheets) — eine Sprache, die für die visuelle Darstellung von Informationen und das Design einer Webseite verantwortlich ist. Stile werden zum Formatieren von Text und einer anderen Stilisierung (Design) von Objekten auf einer Webseite verwendet. CSS wird normalerweise mit gepaart HTML.

JavaScript

Sprache zum Programmieren von Webseiten

JavaScript (JS) - programmiersprache, die dritte Komponente (zusammen mit HTML und CSS) standardmäßiger Webtechnologien, wird zur Erstellung von Websites verwendet. JavaScript macht statische Webseiten dynamisch, flüssig und benutzerfreundlich. Die Programmiersprache JavaScript verwaltet Medien, animiert Bilder, ändert Text auf einer Webseite und mehr.

Lernen JAVASCRIPT JS Referenz

JavaScript Beispiel:

<button onclick="myFunction()">Click Me!</button>

<script>
function myFunction() {
  var x = document.getElementById("demo");
  x.style.fontSize = "25px";

</script>

PHP

Programmiersprache für Webserve

PHP (Hypertext Preprocessor) ist eine universelle Skriptsprache, die zur Entwicklung von Webanwendungen verwendet wird. Sie wird von der überwiegenden Mehrheit der Hosting-Anbieter unterstützt und ist eine der Hauptsprachen für die Erstellung dynamischer Websites. PHP wird oft verwendet, um ein CMS zu erstellen – ein Content-Management-System (Site Engine). PHP läuft auf der Serverseite (Backend) und sorgt für Benutzerinteraktion. Websites wie Facebook, Yahoo, Wikipedia, die WordPress-Engine usw. wurden in PHP erstellt. Die Programmiersprache PHP können Sie auf der Website W3Schools in deutscher Sprache online lernen.
