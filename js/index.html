Навіщо вивчати JavaScript?

JavaScript - одна з трьох технологій, які мають вивчити і знати всі веб-розробники:

   1. HTML для визначення змісту веб-сторінок

   2. CSS для створення макету веб-сторінок

   3. JavaScript для програмування поведінки веб-сторінок

Примітка. Веб-сторінки - це не єдине застосування, де використовується JavaScript. Багато настільних і серверних програм використовують JavaScript. Платформа Node.js є найвідомішим застосуванням для виконання високопродуктивних мережевих додатків. Деякі бази даних, такі як MongoDB і CouchDB, також використовують JavaScript в якості мови програмування.

Що необхідно знати про JavaScript?

JavaScript та Java - це абсолютно різні мови програмування, як по концепції, так і по дизайну.

Мову JavaScript було створено Бренданом Айхом в 1995 році, яка стала стандартом ECMA в 1997 році. ECMA-262 є офиційною назвою стандарту. ECMAScript - це офіційна назва мови JavaScript.

Ви можете прочитати більше про різні версії JavaScript в розділі JS Версії.
