JavaScript kann HTML-Inhalte ändern

Eine von vielen JavaScript-HTML-Methoden ist getElementById().

Das folgende Beispiel „findet“ ein HTML-Element (mit id="demo") und ändert den Elementinhalt (innerHTML) in "Hello JavaScript":

Beispiel

document.getElementById("demo").innerHTML = "Hello JavaScript";
Try it Yourself »

JavaScript akzeptiert sowohl doppelte als auch einfache Anführungszeichen:

Beispiel

document.getElementById('demo').innerHTML = 'Hello JavaScript';
Try it Yourself »

JavaScript kann HTML-Stile (CSS) ändern

Das Ändern des Stils eines HTML-Elements ist eine Variante der Änderung eines HTML-Attributs:

Beispiel

document.getElementById("demo").style.fontSize = "35px";
Try it Yourself »

JavaScript kann HTML-Elemente ausblenden

Das Ausblenden von HTML-Elementen kann durch Ändern des Anzeigestils erfolgen:

Beispiel

document.getElementById("demo").style.display = "none";
Try it Yourself »

JavaScript kann HTML-Elemente anzeigen

Das Anzeigen versteckter HTML-Elemente kann auch durch Ändern des display-Stils erfolgen:

Beispiel

document.getElementById("demo").style.display = "block";
Try it Yourself »

Wussten Sie schon?

JavaScript und Java sind völlig unterschiedliche Sprachen, sowohl im Konzept als auch im Design.

JavaScript wurde 1995 von Brendan Eich erfunden und wurde 1997 zum ECMA-Standard.

ECMA-262 ist der offizielle Name des Standards. ECMAScript ist der offizielle Name der Sprache.
